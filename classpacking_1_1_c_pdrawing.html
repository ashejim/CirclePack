Detailed Description

This class maintains 'packImage', an image buffer, and knows.

This class maintains 'packImage', an image buffer, and knows how to draw lines, circles, polygons, etc., maintain info on colors, real world coords, graphics.

In process of splitting off GUI functions like mouse, drop, and menu events, into CPcanvas, but leaving buffered image here for use with either GUI version or standalone version. Things marked "GUI" will eventually be moved elsewhere.

In 'CirclePack', we have 'PackData' for a circle packing associated with this CPdrawing (in fact, packData is created in this constructor). There are many things that could logically go either in PackData or CPcanvas; in general, try to put things here if they might be used in non-packing situations. Examples are colors, display flag options for circles/faces, opacity. Geometry will be kept in PackData,

CirclePack will have 'NUM_PACKS' (currently 3) CPdrawing's, each managing one buffered image, and it tracks which is 'active'.

In the GUI version, the buffered image is used in the large canvas as well the small images in the control window (see 'ActiveScreen').
