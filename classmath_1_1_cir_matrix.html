Detailed Description

Hermitian 2 × 2 matrix representation of a circle or line in the extended complex plane.

A circle or line can be written as A|z|² + Bz + B̄z̄ + g = 0, where A = ±1 for a circle and A = 0 for a straight line. This equation is encoded as a 2 × 2 Hermitian matrix [A, B; B̄, g]. This class extends Mobius because the image of a circle under a Möbius transformation M can be computed via matrix conjugation: C' = Gᵀ C G̅, where G = M⁻¹·det(M).

Circle case (A = ±1)
For a circle with center z₀ and radius r:
  • B = −z̄₀
  • g = |z₀|² − r²
  • A = +1 means the interior is the disc; A = −1 means the exterior.
Line case (A = 0)
After normalizing so |B| = 1:
  • n = B̄ is the unit normal pointing toward the "interior."
  • −g·n/2 is a point on the line.
  • g > 0 means the origin is interior; g < 0 means exterior.
Note
For drawing purposes, straight lines are stored in CircleSimple objects as very large circles (radius near allMains.CPBase#FAUX_RAD) with lineFlag = true.

CirMatrix math.CirMatrix.applyTransform ( Mobius M,
CirMatrix C,
boolean oriented )
static

For C a 2x2 'CirMatrix', this returns its image under M (oriented true) or M^{-1} (oriented false). Returns a 2x2 'CirMatrix' with the normalization conventions.

The computation is: result = G^{t}*C*conj(G), where G = M^{-1}*det(M) Note: If M = [a b;c d], then G=[d -b;-c a] and G^{t}=[d -c;-b a] (not the hermitian transpose).
