Detailed Description

that displays XMD script preview images in tool tips when hyperlinks to XMD

PreviewImageHyperlinkListener is a HyperlinkListener that displays XMD script preview images in tool tips when hyperlinks to XMD scripts are moused over. Typically, an instance of this class would be added as a HyperlinkListener to a JEditorPane that contains hyperlinks to XMD scripts.
