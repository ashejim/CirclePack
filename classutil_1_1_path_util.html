Detailed Description

Utility routines for working with 'Path2D.

Utility routines for working with 'Path2D.Double' objects: finding length, max distance from the origin, output for postscript, scaling, etc. Creating closed circle paths,

These are not so sophisticated. A key is 'getPathIterator'. If we get a 'FlatteningPathIterator', it gives lists of points to give a polygonal path approximating the path; we will just guess at a 'flatness' parameter to bound how far it drifts from the actual path.
