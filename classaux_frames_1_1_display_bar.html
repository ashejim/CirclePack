Detailed Description

This JPanel contains a vertex label, a bar, and possibly an icon.

This JPanel contains a vertex label, a bar, and possibly an icon. If mode=true, then this represents radius info and the height of bar can be adjusted with the mouse. If mode=false, this is angle sum info provided from outside. the bar color and the icon placement are controlled from outside.
