\doxysubsection{Detailed Description}
This code was started by John Bowers in our work on 3D printing. 

This code was started by John Bowers in our work on 3D printing. It does the "{}field-\/based smoothing"{} step in our infill pattern work, which modifies an existing packing by moving centers in response to some combination of the usual packing criterion and the effects of an ambient field, such as a stress field.

This normally is created in concert with a \textquotesingle{}\doxylink{classmicro_lattice_1_1_micro_grid}{Micro\+Grid}\textquotesingle{} \doxylink{classpacking_1_1_pack_extender}{Pack\+Extender}, \textquotesingle{}parent\+MG\textquotesingle{} because of the access to the ambient field and the path. Commands may then be sent via the \textquotesingle{}Micr\+Grid\textquotesingle{} parent. However, even then, it is associated with a separate packing obtained in \doxylink{classmicro_lattice_1_1_micro_grid}{Micro\+Grid}\textquotesingle{}s "{}tailored sectioning"{} step.

If \textquotesingle{}parent\+MG\textquotesingle{} is null, then the packing must be euclidean. Intensity defaults to 1.\+0 everywhere, the path defaults to \textquotesingle{}\doxylink{classall_mains_1_1_c_p_base_a23c728f5a90fcc5cafd5a75ddeae6ccb}{CPBase.\+Closed\+Path}\textquotesingle{} or to unit circle, min/\+max radii are taken from the packing.
