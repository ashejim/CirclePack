\doxysubsection{Detailed Description}
\doxylink{classgraph_objects_1_1_c_p_circle}{CPCircle} extends Complex. 

\doxylink{classgraph_objects_1_1_c_p_circle}{CPCircle} extends Complex. x,y,radius set from outside. \textquotesingle{}draw\+It(boolean fill)\textquotesingle{} is main call for filled or open. Color settings in graphic context handled elsewhere.

Spherical data is adjusted by parent to act as though sphere is in standard orientation (i.\+e., looking straight down x-\/axis toward origin, z-\/axis vertical).

NOTE:\+ as of 12/\+2009, I\textquotesingle{}ve implemented use of \textquotesingle{}Ellipse2\+D.\+Double\textquotesingle{}. Because of a bug involving \textquotesingle{}Affine\+Transform\textquotesingle{}, I had to change all my calls to use pixel (instead of real) data in the calls. This avoids building my own lists for circles. The old code is in \textquotesingle{}\doxylink{_circle_list_8java}{Circle\+List.\+java}\textquotesingle{}.
