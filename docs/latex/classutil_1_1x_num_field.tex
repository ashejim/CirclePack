\doxysubsection{Detailed Description}
A JText\+Field for entering/\+displaying formated real. 

A JText\+Field for entering/\+displaying formated real values. Can specify 3 to 15 digit length (default 8) and specify a title that goes above the field. If no title is given, this just serves to format a number. This field allows formulas in \$..\$ format.
