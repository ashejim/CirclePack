\doxysubsection{Detailed Description}
This is a new approach to "{}chaperone"{} generalized branch. 

This is a new approach to "{}chaperone"{} generalized branch points. The old approach in \textquotesingle{}Chap\+Branch\+Pt\textquotesingle{} excised a flower, then manipulated it separately from the parent packing. In this new approach, we will modify the parent packing itself in a way that can be undone.

The parameters are the "{}jump"{} circles w1/\+w2 and associated overlap angles.

Construction goes like this:\+ (1) split\+\_\+flower of my\+Index, creating sister2 and sister\+Edge=\texorpdfstring{$<$}{<}my\+Index,sister2\texorpdfstring{$>$}{>}. Petals of sister2 go from w1 to pre\+Jump\+[2]\+, petals of my\+Index go from we to pre\+Jump\+[1]\+. (2) set chap\+Edge\+[1]\+=\texorpdfstring{$<$}{<}sister2,w1\texorpdfstring{$>$}{>}, then split\+\_\+edge; end up with chap\+Edge\+[1]\+=\texorpdfstring{$<$}{<}sister2,chap1\texorpdfstring{$>$}{>} (3) set chap\+Edge\+[2]\+=\texorpdfstring{$<$}{<}my\+Index,w2\texorpdfstring{$>$}{>}, then split\+\_\+edge; end up with chap\+Edge\+[2]\+=\texorpdfstring{$<$}{<}my\+Index,chap2\texorpdfstring{$>$}{>} (4) split\+\_\+edge sister\+Edge to create new\+Br\+Spot.

Delete by reversing:\+ melt\+\_\+edge in this order sister\+Edge -\/\texorpdfstring{$>$}{>} chap\+Edge\+[2]\+ -\/\texorpdfstring{$>$}{>} chap\+Edge\+[1]\+ -\/\texorpdfstring{$>$}{>} sister\+Edge

The overlaps o1/\+o2 are assigned to cclw spokes from chap1/\+2 to pre\+Jump\+[1]\+/\+\+[2]\+, and complementary angles to edges next cclw spokes.
