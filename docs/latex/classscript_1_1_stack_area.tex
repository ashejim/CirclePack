\doxysubsection{Detailed Description}
This is the base panel containing all the \textquotesingle{}stack\+Box\textquotesingle{} objects which. 

This is the base panel containing all the \textquotesingle{}stack\+Box\textquotesingle{} objects which display script elements. There are basically only three objects directly added to this in \textquotesingle{}Pack\+Control.\+init\+Script\+Area\textquotesingle{}:\+ the \textquotesingle{}script\+SBox\textquotesingle{} containing CPScript elements, a rigid spacer, and the \textquotesingle{}data\+SBox\textquotesingle{} containing CPData elements (i.\+e., files).
