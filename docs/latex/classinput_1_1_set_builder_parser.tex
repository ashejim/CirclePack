\doxysubsection{Detailed Description}
This is a parser for set-\/builder type expressions describing. 

This is a parser for set-\/builder type expressions describing sets of objects (circles, faces, tiles, or edges (??)) in \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}:\+ e.\+g. \{c:\+d.\+eq.\+5\} lists verts of degree 5. This class is created when a \{..\} string occurs among the flags in a command (note that commands can also be enclosed with braces if we want a return value). This parser builds a vector of \textquotesingle{}\doxylink{classutil_1_1_select_spec}{Select\+Spec}\textquotesingle{}s. Calling routine must check \textquotesingle{}status\textquotesingle{} and \textquotesingle{}pack\+Data\textquotesingle{} before using the results.

Form is very rigid; here are six main portions:\+
\begin{DoxyEnumerate}
\item Outer curly brackets (this is how we identify the initial string)
\item Target object:\+ v (or c) circles, f faces, t tiles (TODO:\+ might add edges, etc.)
\item Target packing -\/p\texorpdfstring{$\ast$}{*} (optional; subscript for \textquotesingle{}pack\+Data\textquotesingle{} ptr) (with \textquotesingle{}c\textquotesingle{} or \textquotesingle{}f\textquotesingle{}) ~\newline

\item \textquotesingle{}:\+\textquotesingle{} separator
\item One or more \textquotesingle{}specifications\textquotesingle{}:\+ these may be unary or have \textquotesingle{}left\textquotesingle{} and \textquotesingle{}right\textquotesingle{} side target quantities separated by some comparison symbols; left or right might be explicit double value.

a. Target quantities:\+

Unary:\+ bdry=b int=i in list=?list (appropriate to target object type, eg \textquotesingle{}vlist\textquotesingle{})

Binary:\+ aim=a degree=d marked=m rad=r tile type = t angle sum=s modulus of center=z (or ze for eucl center) plot\+\_\+flag=x (face or vert, as appropriate) X(resp. Y,Z)-\/coord in xyz data=X(resp. Y,Z) eucl\+\_\+ratio(p,q)=epq hyp\+\_\+ratio(p,q)=cpq

b. Condition:\+ =, ==, =\texorpdfstring{$<$}{<}, \texorpdfstring{$<$}{<}=, \texorpdfstring{$<$}{<}, \texorpdfstring{$>$}{>}=, =\texorpdfstring{$>$}{>}, \texorpdfstring{$>$}{>} (old style) (new:\+ .eq., .le., .lt., .ge., .gt., .ne.)
\item Connectives between selection specs:\+ \&\&, \texorpdfstring{$\vert$}{|}\texorpdfstring{$\vert$}{|} (inclusive or), or ! (new:\+ .and., .or., .not.)
\end{DoxyEnumerate}

(Note:\+ multiple specifications are taken first-\/to-\/last in order, without any logical grouping -- not syntactically general.)

See \textquotesingle{}util.\+Select\+Spec.\+note\+\_\+to\+\_\+value\textquotesingle{} for retreiving values in processing specifications.
