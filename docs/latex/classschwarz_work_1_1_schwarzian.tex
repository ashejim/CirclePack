\doxysubsection{Detailed Description}
Static routines for computing and manipulating discrete \doxylink{classschwarz_work_1_1_schwarzian}{Schwarzian} derivatives on circle packings. 

The discrete \doxylink{classschwarz_work_1_1_schwarzian}{Schwarzian} derivative is associated with maps between circle packings sharing the same combinatorics. For each oriented edge, it measures how the cross-\/ratio of the four relevant circles changes under the map — analogous to the classical \doxylink{classschwarz_work_1_1_schwarzian}{Schwarzian} derivative \{f, z\} = f\textquotesingle{}\textquotesingle{}\textquotesingle{}/\+f\textquotesingle{} − (3/\+2)(f\textquotesingle{}\textquotesingle{}/\+f\textquotesingle{})² for analytic functions.

This class also handles "{}intrinsic"{} Schwarzians, which are real numbers stored in {\ttfamily \doxylink{}{combinatorics.\+komplex.\+Half\+Edge\+\#schwarzian}} (default 0.\+0) and encode how an individual packing deviates from a reference configuration — the "{}base equilateral"{} (tangent triple of circles with radius √3, symmetric about the origin, first edge vertical through z = 1).
