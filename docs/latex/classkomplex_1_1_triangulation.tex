\doxymbox{\hyperlink{classkomplex_1_1_triangulation_a774a87240898308648634a15f31c41d5}{Triangulation}} komplex.\+Triangulation.\+read\+Tri\+File (\begin{DoxyParamCaption}\item[{Buffered\+Reader}]{fp}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}



Read a file of combinatorial polygons (n-\/tuples of indices, one n-\/tuple per line) and create a \textquotesingle{}\doxylink{classkomplex_1_1_triangulation}{Triangulation}\textquotesingle{} object. Try to handle several possible formats:\+ \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}, OFF, VTK, and (for triangulations) even more rudimentary formats. Data may include 2D or 3D locations as well, and possibly color codes, hence all the complications. Ignore empty and \textquotesingle{}\#\textquotesingle{} comment lines. 

First hurdle is to get beyond the header lines (if any) and other data, to determine the type of file. Then we look for combinatorics:\+ find by key word or line starting with integer, then reading succeeding lines until non-\/integer is encountered. Likewise for XY\+[Z]\+ data:\+ find by key work or line starting with double and continuing until non doubel is encountered. (We ignore indications of how may vertices or faces there are.)

Create a \textquotesingle{}\doxylink{classkomplex_1_1_triangulation}{Triangulation}\textquotesingle{} object from the results, breaking n-\/gons into n triangles when n\texorpdfstring{$>$}{>}3. Include any xy\+[z]\+ infor, possibly color info, and \textquotesingle{}\doxylink{classlist_manip_1_1_vertex_map}{Vertex\+Map}\textquotesingle{} \texorpdfstring{$<$}{<}new,old\texorpdfstring{$>$}{>}.

Here are possible formats:\+

\DoxyHorRuler{0}
 \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} possibilities:\+
\begin{DoxyItemize}
\item \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} format:\+ \textquotesingle{}TRIANGULATION:\+ \{N), N=number of faces.
\item \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} format:\+ \textquotesingle{}CHECKCOUNT:\+\textquotesingle{} look for \textquotesingle{}FACE\+\_\+\+TRIPLES:\+\textquotesingle{} and \textquotesingle{}POINTS:\+\textquotesingle{}
\end{DoxyItemize}

\DoxyHorRuler{0}
 OFF format:\+ look for \textquotesingle{}OFF\textquotesingle{} in first line

OFF \{V\} \{F\} \+[\{E\}]\+ (we ignore) x\+\_\+0 y\+\_\+0 z\+\_\+0 \+[r g b a]\+ real x, y, z (note:\+ indexing starts at zero) .... x\+\_\+\{V-\/1\} y\+\_\+\{V-\/1\} z\+\_\+\{V-\/1\} \{now the faces\} n v\+\_\+1 v\+\_\+2 ..... v\+\_\+n \+[r g b a]\+ ~\newline
 .... k v\+\_\+1 ..... v\+\_\+k

Note:\+ r, g, b, a are doubles; multiply by 255 and cast to integers. Note:\+ In reading face lines, if line item count == 3, we deduce n=3 \hypertarget{classkomplex_1_1_triangulation_autotoc_md2}{}\doxysubsection{\texorpdfstring{and rgba data must be missing.}{and rgba data must be missing.}}\label{classkomplex_1_1_triangulation_autotoc_md2}
VTK format:\+ legacy ascii version starts with \textquotesingle{}\# vtk\textquotesingle{} in first line\hypertarget{classkomplex_1_1_triangulation_autotoc_md3}{}\doxysubsection{\texorpdfstring{vtk Data\+File Version x.\+x}{vtk Data\+File Version x.\+x}}\label{classkomplex_1_1_triangulation_autotoc_md3}
.. POINTS N (or POINT\+\_\+\+DATA N) x y \+[z]\+ ... POLYGONS M (or CELL\+\_\+\+DATA M) n v0 v1 v2 ...\hypertarget{classkomplex_1_1_triangulation_autotoc_md4}{}\doxysubsection{\texorpdfstring{Note:\+ Newer vtk format may be different}{Note:\+ Newer vtk format may be different}}\label{classkomplex_1_1_triangulation_autotoc_md4}
Generic:\+ (none of the above), may get exception
\begin{DoxyItemize}
\item If first line is non-\/numeric, throw exception.
\item If first line is \texorpdfstring{$<$}{<}=2 integers, disregard (vertices/\+faces)
\item If first line is three integers, take as triangular face and continue reading triangular faces.
\item If first line starts with a double, read lines as xy or xyz data.
\item Then look for integer data and use line item count:\+ if 3, then assume triangules, else assume form n v0 v1 ... vn
\end{DoxyItemize}

Post processing:\+ after data is read, we may have adjustments:\+

Any n-\/gons, n\texorpdfstring{$>$}{>}3, are broken into n triangles by creating a barycenter vert (and possibly setting its coords and colors)

Indices may start at 0 and/\+or be non-\/consecutive:\+ we adjust indices, keeping them in order, to start at index 1. Translations kept in \textquotesingle{}tri\+Vertex\+Map\textquotesingle{}.

Triangles might not be compatibly oriented, etc.; this ~\newline
 should be rectified later by user call to \textquotesingle{}parse\+\_\+triangles\textquotesingle{}.
