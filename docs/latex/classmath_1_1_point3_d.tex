\doxysubsection{Detailed Description}
A mutable 3-\/D point /\+ vector with conversions to spherical coordinates. 

Represents a point (x, y, z) in Euclidean 3-\/space. In the context of \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} this class serves a dual purpose:\+
\begin{DoxyItemize}
\item As a plain 3-\/D vector with the usual linear-\/algebra operations (dot product, cross product, projection, normalization).
\item As a point on the Riemann sphere (unit sphere), with conversions between Cartesian (x, y, z) and spherical coordinates (theta, phi).
\end{DoxyItemize}

\begin{DoxyParagraph}{Spherical-\/coordinate convention}
Throughout \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} the spherical coordinates are:\+
\begin{DoxyItemize}
\item {\bfseries{theta}} — azimuthal angle in the xy-\/plane, measured from the positive x-\/axis (range (−π, π]\+).
\item {\bfseries{phi}} — polar angle from the positive z-\/axis (north pole), so phi = 0 is the north pole and phi = π is the south pole.
\end{DoxyItemize}
\end{DoxyParagraph}
The conversion is x = sin(phi)·cos(theta), y = sin(phi)·sin(theta), z = cos(phi).

\begin{DoxyNote}{Note}
The fields are public and mutable.
\end{DoxyNote}
