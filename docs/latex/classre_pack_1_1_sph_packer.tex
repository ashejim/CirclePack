\doxysubsection{Detailed Description}
Spherical circle packing computations. 

Spherical circle packing computations. Currently this is only for maximal packing of a sphere.

As of 3/\+2023, changing from using euclidean packing to hyperbolic due to layout error with the former:\+ We "{}puncture"{} a vertex, max\+\_\+pack in the hyp plane, and project to the sphere, which also normalizes.
