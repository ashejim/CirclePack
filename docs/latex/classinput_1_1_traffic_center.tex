\doxysubsection{Detailed Description}
Controls execution traffic for \textquotesingle{}\doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}\textquotesingle{}. 

Controls execution traffic for \textquotesingle{}\doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}\textquotesingle{}. Intended as a a thread handling essentially all execution calls so GUI can continue operating on its thread. Handles:\+ 1) immediate GUI, housecleaning, etc. (where efficiency important) 2) Main work, pick off individual commands from strings, and send them to \textquotesingle{}\doxylink{classinput_1_1_command_str_parser_ad2c354ce742f567dba1b7cd963fb0323}{Command\+Str\+Parser.\+jexecute}\textquotesingle{}.

Preprocessing includes (can be sensitive to order of processing and must avoid infinite recursion):\+


\begin{DoxyItemize}
\item resolving \textquotesingle{}script\textquotesingle{} named commands
\item finding double quoted strings
\item processes "{}-\/p?"{} packing number flags
\item processes "{}:\+="{} variable assignment
\item processes "{}\+\_\+\texorpdfstring{$\ast$}{*}"{} variable substitution
\item processes !! repeats, recursive calls
\item catches \textquotesingle{}\doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}\textquotesingle{} "{}for/\+\+FOR"{} loops
\item handles "{}delay"{} calls (in appropriate thread)
\item manages \textquotesingle{}\doxylink{classpacking_1_1_pack_extender}{Pack\+Extender}\textquotesingle{} handoffs
\end{DoxyItemize}

3) "{}background"{} thread (or those likely to last long?) Might want separate progress indicator? stop button?

NOTE:\+ Lots of issues on self/\+cross-\/reference, sequence of processing, sequence of execution, result/\+error reporting, history, etc.

Wrapper to handle "{}if-\/then-\/else"{} commands. The functionality is basic, not sophisticated. The form is:\+ IF $<$condition$>$ THEN $<$cmds$>$ ELSE $<$cmds$>$. 

The $<$condition$>$ is set by the string between IF and THEN and should contain no \textquotesingle{};\textquotesingle{}. It must evaluate to true/\+false, possibly with reference to the active pack and static variables and commands that return some value, e.\+g. a boolean or a double.

THEN is followed by a named command \+[.]\+. ELSE is optional and also followed by a named command; default is to issue a \textquotesingle{}break\textquotesingle{}.

These structures can be nested only to the extent that the named commands might themselves contain IF-\/\+THEN-\/\+ELSE.
