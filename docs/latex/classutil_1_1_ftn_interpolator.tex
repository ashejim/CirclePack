\doxysubsection{Detailed Description}
Routines for interpolating a real or complex function. 

Routines for interpolating a real or complex function defined on the unit circle, \+[0,2pi]\+. We are given a vector of values of s (closed up) and an associated vector of values f(s). Given s in \+[0,2pi]\+, we want to linearly interpolate to get f(s). \begin{DoxyAuthor}{Author}
