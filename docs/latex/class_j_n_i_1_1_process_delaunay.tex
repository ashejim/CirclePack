\doxysubsection{Detailed Description}
Static routines for creating Delaunay triangulations with. 

Static routines for creating Delaunay triangulations with calls to \textquotesingle{}qhull\textquotesingle{} (for the sphere) and \textquotesingle{}triangle\textquotesingle{} for regions in the plane. I\textquotesingle{}m using \textquotesingle{}Process\+Builder\textquotesingle{}. The idea is to put the qhull executable in \doxylink{classall_mains_1_1_c_p_base_a468c51605fd06db31805596af65e1766}{CPBase.\+Temp\+Directory} when the program first loads. For a run, the data is placed into a \textquotesingle{}\doxylink{class_j_n_i_1_1_delaunay_data}{Delaunay\+Data}\textquotesingle{} object, the code is run, and the output triangulation is put back into \textquotesingle{}\doxylink{class_j_n_i_1_1_delaunay_data}{Delaunay\+Data}\textquotesingle{}. \begin{DoxyAuthor}{Author}
