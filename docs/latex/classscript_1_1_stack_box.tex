\doxysubsection{Detailed Description}
Our script display will be a Scroll\+Box containing a stack of \doxylink{classscript_1_1_stack_box}{Stack\+Box}\textquotesingle{}s;. 

Our script display will be a Scroll\+Box containing a stack of \doxylink{classscript_1_1_stack_box}{Stack\+Box}\textquotesingle{}s; certain \doxylink{classscript_1_1_stack_box}{Stack\+Box} can contain more \doxylink{classscript_1_1_stack_box}{Stack\+Box}\textquotesingle{}s. \doxylink{classscript_1_1_stack_box}{Stack\+Box}\textquotesingle{}s are panels, with layout depending on type and edit/\+display status. Subclasses are defined for CPscript, CPdata, Section, cmd, file, and text (often implicit) nodes; leaf nodes are cmd, file, and text and don\textquotesingle{}t have children. Currently (8/\+10) am trying to incorporate \textquotesingle{}\doxylink{classscript_1_1_line_s_box}{Line\+SBox}\textquotesingle{}s, which would simply consolidate strings of successive iconified \textquotesingle{}\doxylink{classscript_1_1_cmd_s_box}{Cmd\+SBox}\textquotesingle{}s; this is not yet completed.

If mode is NEW, an appropriate \doxylink{classscript_1_1_c_p_tree_node}{CPTree\+Node} must be created as first thing.
