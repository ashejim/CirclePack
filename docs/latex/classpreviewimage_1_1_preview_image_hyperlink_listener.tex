\doxysubsection{Detailed Description}
that displays XMD script preview images in tool tips when hyperlinks to XMD 

{\ttfamily \doxylink{classpreviewimage_1_1_preview_image_hyperlink_listener}{Preview\+Image\+Hyperlink\+Listener}} is a {\ttfamily Hyperlink\+Listener} that displays XMD script preview images in tool tips when hyperlinks to XMD scripts are moused over. Typically, an instance of this class would be added as a {\ttfamily Hyperlink\+Listener} to a {\ttfamily JEditor\+Pane} that contains hyperlinks to XMD scripts.
