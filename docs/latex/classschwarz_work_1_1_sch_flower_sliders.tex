\doxysubsection{Detailed Description}
A Frame for sliders controlling specified. 

A Frame for sliders controlling specified intrinsic schwarzians s for flowers -- see "{}flower"{} command in \textquotesingle{}\doxylink{_schwarz_map_8java}{Schwarz\+Map.\+java}\textquotesingle{}. Note:\+ often computations use the u-\/variable, u=1-\/s, see "{}\+Sch\+Flower\+Data"{}, but these sliders controls the s\textquotesingle{}s themselves.

Display will also so the "{}errors"{} in the effect of the current schwarzians, see \textquotesingle{}\doxylink{classschwarz_work_1_1_sch_flower_data}{Sch\+Flower\+Data}. eucl\+Flower\textquotesingle{}.

CAUTION:\+ the \textquotesingle{}errors\textquotesingle{} depend on layout process:\+ we start layout using the edge schwarzian closest to the uniform for an n-\/flower. Starting with a different edge could give different errors. Fortunately, error = 0 is independent of layout process.
