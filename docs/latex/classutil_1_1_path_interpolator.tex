Here a "{}path"{} means a polygonal path in one of the geometries,. 

Here a "{}path"{} means a polygonal path in one of the geometries, parameterized by arc length. It may be closed or open. Stored using \+[]\+complex for the path\textquotesingle{}s vertex locations and \+[]\+double for the corresponding arclength parameter values.
