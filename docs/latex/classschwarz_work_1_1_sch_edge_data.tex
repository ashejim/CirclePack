\doxysubsection{Detailed Description}
Data for \textquotesingle{}\doxylink{classcombinatorics_1_1komplex_1_1_half_edge}{Half\+Edge}\textquotesingle{}s subject to adjustment. 

Data for \textquotesingle{}\doxylink{classcombinatorics_1_1komplex_1_1_half_edge}{Half\+Edge}\textquotesingle{}s subject to adjustment when circle packing via intrinsic schwarzians. Computing a new schwarzian involves the N-\/3 previous edge schwarzians, so we want that to be easy to compute.

These objects are instantiate when \textquotesingle{}\doxylink{classschwarz_work_1_1_schwarz_pack}{Schwarz\+Pack}\textquotesingle{} is initiated and put in an array, indexed starting at 1. See \textquotesingle{}the\+Edges\textquotesingle{} for array index.
