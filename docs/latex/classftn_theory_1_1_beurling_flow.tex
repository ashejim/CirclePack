\doxysubsection{Detailed Description}
Routine for experimenting with the Beurling-\/\+Riemann Mapping. 

Routine for experimenting with the Beurling-\/\+Riemann Mapping Theorem. (An earlier version was \textquotesingle{}Beurling\+\_\+flow.\+java\textquotesingle{}.) The goal is to use an ambient function Phi, continuous and (typically) non-\/zero on the plane.

Idea is to set up iterative process. Beurling\textquotesingle{}s generalization of Riemann mapping theorem says that given bounded, continuous positive function h on the plane, there exists a univalent analytic function f so that \texorpdfstring{$\vert$}{|}f\textquotesingle{}(z)\texorpdfstring{$\vert$}{|} -\/ h(f(z)) -\/-\/\texorpdfstring{$>$}{>} 0 as z -\/-\/\texorpdfstring{$>$}{>} e\texorpdfstring{$^\wedge$}{\string^}\{it\} (on unit circle), f(0)=0 and f\textquotesingle{}(0)\texorpdfstring{$>$}{>}0. In some cases, f is unique. Wegert, Roth, et al have extended this to allow branching as well.
