\doxysubsection{Detailed Description}
\doxylink{classutil_1_1_complex_field}{Complex\+Field} is a panel-\/contained text field for displaying complex. 

\doxylink{classutil_1_1_complex_field}{Complex\+Field} is a panel-\/contained text field for displaying complex values and optionally includes a text label title. It is intended as a re-\/sizable replacement for the {\ttfamily \doxylink{classutil_1_1z_num_field}{z\+Num\+Field}} class.
