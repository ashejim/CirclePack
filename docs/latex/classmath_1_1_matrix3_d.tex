\doxysubsection{Detailed Description}
3 × 3 real matrix for rotations, rigid motions, and general linear algebra on 3-\/D vectors. 

In \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} this class is used primarily for SO(3) rotation matrices that orient the "{}apparent sphere"{} (the viewer\textquotesingle{}s perspective of the Riemann sphere). For an orthogonal matrix, the inverse is simply the transpose.

\begin{DoxyParagraph}{Coordinate-\/axis rotation conventions (right-\/handed system)}

\begin{DoxyPre}
  RX =  1       0        0        (ccw in yz-\/plane)
        0     cos t   −sin t
        0     sin t    cos t

  RY =  cos t    0     sin t      (ccw in zx-\/plane)
          0      1       0
       −sin t    0     cos t

  RZ =  cos t  −sin t    0        (ccw in xy-\/plane)
        sin t   cos t    0
          0       0      1
\end{DoxyPre}

\end{DoxyParagraph}
