\doxysubsection{Detailed Description}
This is an abstract class encapsulating the various mechanisms for. 

This is an abstract class encapsulating the various mechanisms for acquisition, execution, and reporting of \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} command strings and for external data sockets. Various sources can be accommodated:\+ \textquotesingle{}\doxylink{classinput_1_1_my_console}{My\+Console}\textquotesingle{}s, sockets, tools, \textquotesingle{}fexec\textquotesingle{} files, etc. (It will be introduced slowly, as time permits and features demand.) \begin{DoxyAuthor}{Author}
