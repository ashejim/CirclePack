\doxysubsection{Detailed Description}
\doxylink{classgraph_objects_1_1_c_p_face}{CPFace}. This is a persistent object, one for each pack; it is. 

\doxylink{classgraph_objects_1_1_c_p_face}{CPFace}. This is a persistent object, one for each pack; it is reused by changing the data as needed. \doxylink{classgraph_objects_1_1_c_p_face}{CPFace} handles triangles (N=3) or more general polygons. The various (x,y) values for vertices are set from outside. \textquotesingle{}draw\+It(boolean fill)\textquotesingle{} is main call for filled or open. Color settings in graphic context are handled elsewhere; generally filled requires parent to make two calls because of different colors.

Spherical data is adjusted by parent to act as though sphere is in standard orientation (i.\+e., looking straight down x-\/axis toward origin, z-\/axis vertical).
