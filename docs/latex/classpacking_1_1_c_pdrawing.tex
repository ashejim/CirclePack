\doxysubsection{Detailed Description}
This class maintains \textquotesingle{}pack\+Image\textquotesingle{}, an image buffer, and knows. 

This class maintains \textquotesingle{}pack\+Image\textquotesingle{}, an image buffer, and knows how to draw lines, circles, polygons, etc., maintain info on colors, real world coords, graphics.

In process of splitting off GUI functions like mouse, drop, and menu events, into CPcanvas, but leaving buffered image here for use with either GUI version or standalone version. Things marked "{}\+GUI"{} will eventually be moved elsewhere.

In \textquotesingle{}\doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}\textquotesingle{}, we have \textquotesingle{}\doxylink{classpacking_1_1_pack_data}{Pack\+Data}\textquotesingle{} for a circle packing associated with this \doxylink{classpacking_1_1_c_pdrawing}{CPdrawing} (in fact, pack\+Data is created in this constructor). There are many things that could logically go either in \doxylink{classpacking_1_1_pack_data}{Pack\+Data} or CPcanvas; in general, try to put things here if they might be used in non-\/packing situations. Examples are colors, display flag options for circles/\+faces, opacity. Geometry will be kept in \doxylink{classpacking_1_1_pack_data}{Pack\+Data},

\doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} will have \textquotesingle{}NUM\+\_\+\+PACKS\textquotesingle{} (currently 3) \doxylink{classpacking_1_1_c_pdrawing}{CPdrawing}\textquotesingle{}s, each managing one buffered image, and it tracks which is \textquotesingle{}active\textquotesingle{}.
