\doxysubsection{Detailed Description}
Computational routines for geometry in the Lorentz (hyperboloid) model. 

Beginning now (12/\+2012) to build computations for geometry based on Lorentzian geometry. I expect to use these in certain situations now and consider using them to replace older routines as I build up the procedures and reliability. The generality, linearity, and associated "{}geometric algebra"{} are advantages. (Probably should have transitioned years ago!)

We model hyperbolic 3-\/space as R\texorpdfstring{$^\wedge$}{\string^}4 with the Lorentz quadratic form Q(t,x,y,z)=t\texorpdfstring{$^\wedge$}{\string^}2-\/x\texorpdfstring{$^\wedge$}{\string^}2-\/y\texorpdfstring{$^\wedge$}{\string^}2-\/z\texorpdfstring{$^\wedge$}{\string^}2 of signature (1,3) and the associated bilinear form (t,x,y,z).(t\textquotesingle{},x\textquotesingle{},y\textquotesingle{},z\textquotesingle{})=tt\textquotesingle{}-\/xx\textquotesingle{}-\/yy\textquotesingle{}-\/zz\textquotesingle{}. NOTE:\+ in the literature, both this Q and its negative, signature (3,1), are used. E.\+g., see papers by Hestenes.

The light cone is the cone of null vectors Q(v)=0. The hyperboloid model of hyperbolic geometry is the sheet S+ of the hyperboloid Q(v)=1, t\texorpdfstring{$^\wedge$}{\string^}2-\/x\texorpdfstring{$^\wedge$}{\string^}2-\/y\texorpdfstring{$^\wedge$}{\string^}2-\/z\texorpdfstring{$^\wedge$}{\string^}2=1, with t\texorpdfstring{$>$}{>}0. The identification is via projection from (-\/1,0,0,0):\+ namely if x\texorpdfstring{$^\wedge$}{\string^}2+y\texorpdfstring{$^\wedge$}{\string^}2+z\texorpdfstring{$^\wedge$}{\string^}2\texorpdfstring{$<$}{<}1, then (x,y,z) \texorpdfstring{$<$}{<}-\/-\/-\/\texorpdfstring{$>$}{>} (s,sx,sy,sz) where s=1/\+sqrt(1-\/x\texorpdfstring{$^\wedge$}{\string^}2-\/y\texorpdfstring{$^\wedge$}{\string^}2-\/z\texorpdfstring{$^\wedge$}{\string^}2). S+ is asymptotic to the light cone as t grows so its boundary is the unit sphere.

Discs are represented as points, namely

\{(t,x,y,z):\+ t\texorpdfstring{$^\wedge$}{\string^}2-\/x\texorpdfstring{$^\wedge$}{\string^}2-\/y\texorpdfstring{$^\wedge$}{\string^}2-\/z\texorpdfstring{$^\wedge$}{\string^}2 = -\/1\}, t\texorpdfstring{$>$}{>}0

Boundary of disc (t,x,y,z) is

\{(a,b,c):\+a\texorpdfstring{$^\wedge$}{\string^}2+b\texorpdfstring{$^\wedge$}{\string^}2+c\texorpdfstring{$^\wedge$}{\string^}2=1, ax+by+cz=t\}

where t/\+(x\texorpdfstring{$^\wedge$}{\string^}2+y\texorpdfstring{$^\wedge$}{\string^}2+z\texorpdfstring{$^\wedge$}{\string^}2)=cos(r), r = sph radius.

For the 2D hyperbolic model simply set z=0.
