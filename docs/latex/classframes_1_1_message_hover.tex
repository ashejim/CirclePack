\doxysubsection{Detailed Description}
Created at startup. 

Created at startup. Provides hovering shell/\+scratch panel. When locked, panes are scrollable and command entry line is added. Hover occurs when mouse is in "{}\+Messages"{} button or \textquotesingle{}Hover\textquotesingle{} buttons in \textquotesingle{}\doxylink{classinput_1_1_my_console}{My\+Console}\textquotesingle{}s.

This class maintains the command history, for usual up/\+down command recall. The shell shows command lines as executed, also messages, errors, etc.
