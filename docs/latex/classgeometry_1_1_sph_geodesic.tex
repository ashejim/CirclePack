\doxysubsection{Detailed Description}
Spherical geodesic data for visual rendering on the apparent sphere. 

Spherical geodesic for use in plotting. Set up data for the visual portion of the geodesic only. CAUTION:\+ positions are relevant to the \textquotesingle{}apparent\textquotesingle{} sphere (depends on \textquotesingle{}spherical view\textquotesingle{}). Geodesic considered \textquotesingle{}visible\textquotesingle{} iff:\+
\begin{DoxyItemize}
\item at least one end is ON\+\_\+\+FRONT
\item it follows horizon counterclockwise
\item its ends are approximately antipodals ON\+\_\+\+HORIZON (so \textquotesingle{}line\+Flag\textquotesingle{} is set and use geodesic towards the viewer). Note that \textquotesingle{}invisible\textquotesingle{} geodesic can start or end ON\+\_\+\+HORIZON 
\end{DoxyItemize}
