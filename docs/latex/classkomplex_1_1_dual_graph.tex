\doxysubsection{Detailed Description}
For creating/\+manipulating combinatoric information for. 

For creating/\+manipulating combinatoric information for the dual graphs G for circle packing complexes K, i.\+e., the vertices of G correspond to faces of K, an edge of G, \texorpdfstring{$<$}{<}f,g\texorpdfstring{$>$}{>} indicates that faces f and g share an edge, and the (polygonal) faces of G correspond to the vertices of K.


\begin{DoxyItemize}
\item Now (2/\+2009) the aim is building spanning trees for G with static methods.
\item Also (6/\+2010) building plottrees, dual trees showing the order in which faces are laid down in drawing order. (Note that a plottree will generally not pick up all the nodes of the dual graph; some nodes are not needed since the three circles of that face have already been laid out by other faces.)
\item Also (12/\+2010), tree\+Trim, which modifies a tree by removing designated edges, and reattaching the cutoff tree by introducing some non-\/proscribed edge. Main idea is to be able to easily modify a plottree to get a new plot tree with particular properties --- eg. to get a particularly nice layout of a torus. \begin{DoxyAuthor}{Author}

\end{DoxyAuthor}

\end{DoxyItemize}
