\doxysubsection{Detailed Description}
\doxylink{classmath_1_1group_1_1_schottky}{Schottky} group implementation using paired circles and Möbius generators.\+he word. 

Title:\+ 

Description:\+ Specifics of the class:\+ Only two generators A, B. 

Words apply only to circles that don\textquotesingle{}t contradict the last letter in the word.

If the word ends with A, we don\textquotesingle{}t apply it to circle a, for instance.
