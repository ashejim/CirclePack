Return a String in response to a query. This may have three parts:\+ \textquotesingle{}words\textquotesingle{}, \textquotesingle{}ans\textquotesingle{}, and \textquotesingle{}suffix\textquotesingle{}. \textquotesingle{}ans\textquotesingle{} may represent (in \textquotesingle{}String\textquotesingle{} form) a value, list, etc, and generally can be used as a variable value in commands. 

If \textquotesingle{}for\+Msg\textquotesingle{} is true, query is intended as a message, so \textquotesingle{}words\textquotesingle{} is prepended. Note that \textquotesingle{}words\textquotesingle{} defaults to "{}query (p\texorpdfstring{$\ast$}{*}) = "{}; may want to change this if it doesn\textquotesingle{}t depend on the current packing (e.\+g., \textquotesingle{}?Vlist\textquotesingle{}).

Also, if \textquotesingle{}for\+Msg\textquotesingle{}, then some lists may be limited to 12 items, e.\+g. if vertex list is longer, put in \textquotesingle{}...\textquotesingle{} via a \textquotesingle{}suffix\textquotesingle{} string. Long strings are likewise truncated.
