\doxysubsection{Detailed Description}
Utility class holding fundamental geometric data needed for. 

Utility class holding fundamental geometric data needed for repacking and other computations, localized to (triangular only) face of some parent complex. Typically used for temporary purposes, so NOT routinely maintained.

The derived class \textquotesingle{}Tri\+Aspects\textquotesingle{} has been used in additional situations.\+e.\+g., with projective and affine structures and with discrete Schwarzians. \textquotesingle{}labels\textquotesingle{} are typically of use only in the eucl setting and are treated as homogeneous radii -- that is, only their ratios are important.
