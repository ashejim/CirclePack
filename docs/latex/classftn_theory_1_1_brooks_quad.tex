\doxysubsection{Detailed Description}
"{}\+Brooks"{} packings are patterns of circles inside quadrilateral interstices. 

"{}\+Brooks"{} packings are patterns of circles inside quadrilateral interstices. A "{}quad"{} interstice is one formed by 4 successively tangent circles with mutually disjoint interiors, labeled T,L,B,R for \textquotesingle{}top\textquotesingle{}, \textquotesingle{}left\textquotesingle{}, \textquotesingle{}bottom\textquotesingle{}, \textquotesingle{}right\textquotesingle{}. As a Brooks pattern develops, there\textquotesingle{}s always a \textquotesingle{}plug\textquotesingle{} vertex P, which is a 4-\/degree vertex whose petals form the current quadrilateral interstice; so P isn\textquotesingle{}t technically a part of the Brooks pattern, just necessary for the mechanics, since we need a packing. Generically, Brooks packings go on ad infinitum, but at each finite stage the plug caps off remaining quad interstice.

This code converts strings of H\textquotesingle{}s and V\textquotesingle{}s into Brooks packings by modifying an initial \textquotesingle{}quad\textquotesingle{} by adding successive \textquotesingle{}horizontal\textquotesingle{} and \textquotesingle{}vertical\textquotesingle{} circles a la Robert Brooks, Duke Math.\+J. 1985. The original routine used here was conceived by Matt Cathey.

There are two modes of operation indicated by flag:\+ (1) insert new circle in location as in Brooks\textquotesingle{} original scheme; (2) insert new circle in alternating fashion:\+ verticals alternate left/\+right, horizontals, up/\+down.
