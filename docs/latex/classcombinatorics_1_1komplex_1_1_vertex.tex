\doxysubsection{Detailed Description}
DCEL vertex holding combinatorial, geometric, and packing data. 

Each vertex in the DCEL stores a pointer to one outgoing half-\/edge ({\ttfamily \doxylink{classcombinatorics_1_1komplex_1_1_vertex_a3f35e7218479a522d513658c94f9f293}{halfedge}}), from which the full star (flower) can be traversed. For boundary vertices, {\ttfamily halfedge} points to the downstream (counter-\/clockwise) boundary edge.

Geometric data includes the circle\textquotesingle{}s center, radius, desired angle sum (aim), and current angle sum (curvature).

\begin{DoxyNote}{Note}
In hyperbolic geometry, radii are stored in "{}x-\/radius"{} form:\+ x = 1 − e\texorpdfstring{$^\wedge$}{\string^}\{−2h\} for finite hyperbolic radius h. For horocycles (h = ∞), the negative of the Euclidean radius is stored for plotting convenience.
\end{DoxyNote}
