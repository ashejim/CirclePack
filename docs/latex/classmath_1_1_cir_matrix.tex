\doxysubsection{Detailed Description}
Hermitian 2 × 2 matrix representation of a circle or line in the extended complex plane. 

A circle or line can be written as A\texorpdfstring{$\vert$}{|}z\texorpdfstring{$\vert$}{|}² + Bz + B̄z̄ + g = 0, where A = ±1 for a circle and A = 0 for a straight line. This equation is encoded as a 2 × 2 Hermitian matrix \+[A, B; B̄, g]\+. This class extends {\ttfamily \doxylink{classmath_1_1_mobius}{Mobius}} because the image of a circle under a Möbius transformation M can be computed via matrix conjugation:\+ C\textquotesingle{} = Gᵀ C G̅, where G = M⁻¹·det(\+M).

\begin{DoxyParagraph}{Circle case (A = ±1)}
For a circle with center z₀ and radius r:\+
\begin{DoxyItemize}
\item B = −z̄₀
\item g = \texorpdfstring{$\vert$}{|}z₀\texorpdfstring{$\vert$}{|}² − r²
\item A = +1 means the interior is the disc; A = −1 means the exterior.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Line case (A = 0)}
After normalizing so \texorpdfstring{$\vert$}{|}\+B\texorpdfstring{$\vert$}{|} = 1:\+
\begin{DoxyItemize}
\item n = B̄ is the unit normal pointing toward the "{}interior."{}
\item −g·n/\+2 is a point on the line.
\item g \texorpdfstring{$>$}{>} 0 means the origin is interior; g \texorpdfstring{$<$}{<} 0 means exterior.
\end{DoxyItemize}
