\doxysubsection{Detailed Description}
Panel containing the active packing canvas and associated. 

Panel containing the active packing canvas and associated tool bars. Size currently fixed. (Note:\+ in old versions search on \textquotesingle{}DISP\+\_\+\+CHOICE\textquotesingle{} to find some former attempts. Some issues:\+


\begin{DoxyEnumerate}
\item Fixed size:\+ (Current setup, 7/\+08) the frame size is \textquotesingle{}CPBase.\+image\+Buffer\+Size\textquotesingle{} (possibly adjusted on startup), so the \textquotesingle{}pack\+Image\textquotesingle{}s, pix sizes, etc., can be fixed. This has display speed advantage, since \textquotesingle{}pack\+Image\textquotesingle{} and \textquotesingle{}active\+Screen\textquotesingle{} sizes are the same, allowing direct copy.
\item Variable size:\+ If the \textquotesingle{}active\+Screen\textquotesingle{} size is allowed to change, there are several adjustments:\+ \textquotesingle{}resize()\textquotesingle{} keeps canvas square, have \textquotesingle{}canvas\+Package\textquotesingle{} as container for \textquotesingle{}active\+Screen\textquotesingle{} because of layout problems, but still can\textquotesingle{}t seem to get \textquotesingle{}active\+Screen\textquotesingle{} and \textquotesingle{}pack\+Image\textquotesingle{} in sync. Size adjustments are slow.
\end{DoxyEnumerate}

There two options for adjusting:\+

2a. Change \textquotesingle{}pack\+Image\textquotesingle{} buffer sizes (for all CPDrawing\textquotesingle{}s) to match \textquotesingle{}active\+Screen\textquotesingle{} size. This requires call to \textquotesingle{}CPDrawing.\+reset\+Canvas\+Size()\textquotesingle{}, which copies \textquotesingle{}pack\+Image\textquotesingle{} to new buffer, etc.. Also have to adjust \textquotesingle{}pix\+Width\textquotesingle{} etc. Displays do not look good when small because of pixelation.

2b. Keep image buffers the same size but repaint using \textquotesingle{}get\+Scaled\+Instance\textquotesingle{} in \textquotesingle{}\doxylink{classcanvasses_1_1_active_wrapper}{Active\+Wrapper}\textquotesingle{}. Pixel sizes don\textquotesingle{}t change, but have to keep \textquotesingle{}Active\+Wrapper.\+display\+Width/\+\+Height\textquotesingle{} updated, and \textquotesingle{}adjust\+Pt\textquotesingle{} routine to coordinate mouse clicks with the pixels. Also, would need to adjust font size. Displays look nice, but repaints are too slow. Also, have to adjust \textquotesingle{}CPDrawing.\+drawX/\+\+YAxis\textquotesingle{} to get the axes right.
