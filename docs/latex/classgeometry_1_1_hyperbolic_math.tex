\doxysubsection{Detailed Description}
Static methods for mathematical operations in hyperbolic geometry (Poincaré disc model). 

Our model of the hyperbolic plane is the unit disc with the Poincaré metric; namely ds=2\texorpdfstring{$\vert$}{|}dz\texorpdfstring{$\vert$}{|}/\+(1-\/z\texorpdfstring{$^\wedge$}{\string^}2), so curvature is -\/1.

Note on radii:\+ h is "{}hyperbolic"{} radius, value the user sets/\+sees, but internal storage is using "{}x-\/radius"{} and on occasion, "{}s-\/radius"{}. Infinite h corresponds with s=0 and x=1.

Some formulae:\+


\begin{DoxyItemize}
\item relations among the h-\/, x-\/, and s-\/radii:\+ s=exp(-\/h), x = 1-\/s\texorpdfstring{$\ast$}{*}s, s=sqrt(1-\/x), h = log(1/\+s) = -\/1/\+2 log(1-\/x)
\item point with eucl distance r and hyp distance h from origin, then:\+ h=log((1+r)/\+(1-\/r)), r=(exp(h)-\/1)/\+(exp(h)+1), s=(1-\/r)/\+(1+r) and cosh(h)=(1+r\texorpdfstring{$\ast$}{*}r)/\+(1-\/r\texorpdfstring{$\ast$}{*}r)
\item for a circle eucl rad r at the origin this means:\+ s = (1-\/r)/\+(1+r), r = (1-\/s)/\+(1+s), x = 4r/\+(r+1)\texorpdfstring{$^\wedge$}{\string^}2 = 4r/\+(1+2r+r\texorpdfstring{$^\wedge$}{\string^}2), and r=(2-\/x -\/ 2sqrt(1-\/x)) /\+ x, h = log((1+r)/\+(1-\/r))
\item Hyperbolic law of cosines:\+ Given hyp side lengths l1, l2, l3, the angle a1 opposite side l1 is a1 = acos( (cosh(l2)\texorpdfstring{$\ast$}{*}cosh(l3) -\/ cosh(l1)) /\+ (sinh(l2)\texorpdfstring{$\ast$}{*}sinh(l3)) ) For infinite lengths, this is ambiguous, but can be resolved from radii if triangle is formed by circles. See \textquotesingle{}h\+\_\+cos\+\_\+s\+\_\+overlap\textquotesingle{} and \textquotesingle{}h\+\_\+comp\+\_\+x\+\_\+cos\textquotesingle{}.
\item If ircle at origin of eucl radius a has inv dist ivd with a horocycle, then the eucl rad of horocycle is given by b b = (1-\/a\texorpdfstring{$\ast$}{*}a)/\+(2+2\texorpdfstring{$\ast$}{*}a\texorpdfstring{$\ast$}{*}ivd)
\item See \textquotesingle{}h\+\_\+to\+\_\+e\+\_\+data\textquotesingle{} and \textquotesingle{}e\+\_\+to\+\_\+h\+\_\+data\textquotesingle{} for converting centers/\+radii
\item hyp distance l between circles of radii h1, h2, and inv distance ivd:\+ cosh(l) = cosh(h1)\texorpdfstring{$\ast$}{*}cosh(h2)+sinh(h1)\texorpdfstring{$\ast$}{*}sinh(h2)\texorpdfstring{$\ast$}{*}ivd length l infinite iff at least one of h1, h2 is infinite. See \textquotesingle{}h\+\_\+invdist\+\_\+length\textquotesingle{} and \textquotesingle{}h\+\_\+dist\textquotesingle{}
\end{DoxyItemize}

In addition to the disc, hyperbolic geometry has the "{}hyperboloid"{} model. Namely, consider the sheet S with t\texorpdfstring{$>$}{>}0 of the hyperboloid t\texorpdfstring{$^\wedge$}{\string^}2-\/(x1\texorpdfstring{$^\wedge$}{\string^}2+x2\texorpdfstring{$^\wedge$}{\string^}2)=1, or t=sqrt(1+(x1\texorpdfstring{$^\wedge$}{\string^}2+x2\texorpdfstring{$^\wedge$}{\string^}2)). Find third circle in triple when first two are horocycles with given centers (on unit circle), three radii (neg eucl for horocycles), along with all inv distances. Want arc (z1,z0) to be considered cclw, so third circle is generally outside the geodesic between them. Note that we make no consistency checks. 

Create \doxylink{classmath_1_1_mobius}{Mobius} M of disc to UHP with a -\/\texorpdfstring{$>$}{>} 0 x -\/\texorpdfstring{$>$}{>} i, b -\/\texorpdfstring{$>$}{>} infinity, where x is the intersection of first (sometimes second) horocycle with the geodesic arc between them. M=\+[i\texorpdfstring{$\ast$}{*}(b-\/x), i\texorpdfstring{$\ast$}{*}a\texorpdfstring{$\ast$}{*}(x-\/b); a-\/x, b\texorpdfstring{$\ast$}{*}(x-\/a)]\+. M moves the unit disc to the UHP; the first horocycle has center at the origin and goes through i (so radius 1/\+2), while the second horocycle is a horizontal line

Computational fact:\+ in UHP, if one horocycle is tangent to real line and has eucl rad R and another is tangent at infinity with inv distance d between them, then second is horizontal line at eucl height (1+d)\texorpdfstring{$\ast$}{*}R. We also use this fact, properly adjusted, when third is not a horocycle.

Under M, first becomes circle through i and tangent at origin, second is horizontal line through at height H = 1+ivd0. If third is horocycle, its eucl rad is R2=H/\+(1+ivd1). If x2\texorpdfstring{$>$}{>}0, then can solve for eucl radius R2 by using overlap ivd1 with horizontal line and using hyperbolic radius h as integral (1/\+t)dt.

In both cases, compute eucl center z2 and radius R2 of third circle. Apply inv(\+M) to get eucl third circle in the disc, and finally, convert to hyperbolic circle.
