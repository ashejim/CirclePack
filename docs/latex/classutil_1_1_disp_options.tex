\doxysubsection{Detailed Description}
\doxylink{classutil_1_1_disp_options}{Disp\+Options} simply holds the default display options for a given circle. 

\doxylink{classutil_1_1_disp_options}{Disp\+Options} simply holds the default display options for a given circle packing and supplies it to \textquotesingle{}disp\textquotesingle{} commands when specific flags are lacking. (\textquotesingle{}disp\textquotesingle{} and \textquotesingle{}disp -\/wr\textquotesingle{} calls in \textquotesingle{}Pack\+Control.\+naked\+Disp\textquotesingle{}). The boolean data elements here correspond with checkboxes in the Display\+Panel, \textquotesingle{}tailored\textquotesingle{} is the user-\/specified text string in Display\+Panel (for advanced users). \begin{DoxyAuthor}{Author}
