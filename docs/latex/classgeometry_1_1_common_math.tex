\doxysubsection{Detailed Description}
Geometry-\/independent dispatch layer for mathematical routines. 

General calls to math routines; these call the relevant routine in the appropriate geometry. I try to get similar names and arguments in every geometry. Only towards the end of this file are routines that require \textquotesingle{}\doxylink{classpacking_1_1_pack_data}{Pack\+Data}\textquotesingle{}. \begin{DoxyAuthor}{Author}
