\doxysubsection{Detailed Description}
Work with intrinsic schwarzians of individual. 

Work with intrinsic schwarzians of individual closed flowers in normalized setting:\+ Center C is upper half plane, petal c\+\_\+n is half-\/plane \{y\texorpdfstring{$<$}{<}= -\/2\}, petal c\+\_\+1 is tangent to C at at 0 with eucl radius 1, petals c\+\_\+j tangent at t\+[j]\+ for j=1,..,n-\/1. Data is \textquotesingle{}uzian\textquotesingle{} variable u, u=1-\/s where s is intrinsic schwarzian. Uzians u should be restricted to be positive.

CAUTION:\+ In my paper on schwarzians, indexing is from zero, e.\+g. uzians are \{u\+\_\+0,u\+\_\+1,...u\+\_\+\{n-\/1\}\} and in normalization, c\+\_\+0 is the half plane y\texorpdfstring{$<$}{<}=-\/2i. However, in the code indexing is from 1, so c\+\_\+0 is c\+\_\+n. However, in both the paper and the code, the n-\/3 parameters are u\+\_\+1,...,u\+\_\+\{n-\/3\}.

Easy to encounter errors or improper values in computations:\+ e.\+g., if some u becomes negative. We throw a \textquotesingle{}\doxylink{classexceptions_1_1_data_exception}{Data\+Exception}\textquotesingle{} (and generally catch this for some tailored reaction).

CAUTION:\+ Major difficulties with branching:\+
\begin{DoxyItemize}
\item methods (e.\+g., function f) for finding the n-\/2 schwarzians only for un-\/branched
\item direction reverses:\+ one step left, then adjust next step to be sure it goes right. Displacement can be negative or zero (if target petal to C at infinity, i.\+e., is a half plane.)
\item If c\+\_\+\{n-\/2\} is a half plane, then cannot complete since c\+\_\+\{n-\/1\} is ambiguous.
\item for single branching, could overcome this by finishing using layout in reverse from c\+\_\+\{n-\/1\}.
\item but this doesn\textquotesingle{}t work if we don\textquotesingle{}t know how to compute the last schwarzian.
\end{DoxyItemize}

Variables are indexed from 1 so they correspond to normal petal indexing from 1 to n, so argument \textquotesingle{}uzs\textquotesingle{} in constructor is length N+1. The last 3 uzians in \textquotesingle{}uzs\textquotesingle{} may be recomputed to complete a packing (edge) label.

Compute full array of "{}constraint"{} functions C\+\_\+j, j=0,...,(N-\/1), based on given N uzians \{u0,u1,...,u(N-\/1)\}. We set C\+\_\+0=0.\+0, C\+\_\+1=1.\+0, and C\+\_\+\{N-\/1\}=1.\+0 in advance and compute the rest recursively, j=2,...,N-\/2. As long as the C\+\_\+k, 2\texorpdfstring{$<$}{<}=k\texorpdfstring{$<$}{<}j, are positive, then C\+\_\+j is the reciprocal root of radius r\+\_\+j, j=2,..., N-\/2, in the normalized layout. Use this recursive:\+ 

(\texorpdfstring{$\ast$}{*}) C\+\_\+\{j+1\}=sqrt\{3\}\texorpdfstring{$\ast$}{*}u\+\_\+j\texorpdfstring{$\ast$}{*}\+C\+\_\+j -\/ C\+\_\+\{j-\/1\} for 1 \texorpdfstring{$<$}{<}= j \texorpdfstring{$<$}{<}= N-\/2.

Convention in the paper on schwarzians is that constraint C\+\_\+j has j-\/1 arguments u1,...,u\{j-\/1\}, for 2\texorpdfstring{$<$}{<}=j\texorpdfstring{$<$}{<}=N-\/2. If some C\+\_\+j is negative, then the flower is branched and (\texorpdfstring{$\ast$}{*}) may not hold, but we continue anyway.

NOTE:\+ Often we call this with uzians indices shifted; calling routine takes care of this on input and output.

NOTE:\+ If C\+\_\+j are positive, the flower is un-\/branched and we can compute everything in the half-\/plane normalization from the C\+\_\+j:\+
\begin{DoxyItemize}
\item r\+\_\+j=1/\+(C\+\_\+j)\texorpdfstring{$^\wedge$}{\string^}2 and
\item displacement delta\+\_\+j=2/\+(C\+\_\+j\texorpdfstring{$\ast$}{*}\+C\+\_\+\{j+1\}), 1\texorpdfstring{$<$}{<}=j\texorpdfstring{$<$}{<}=N-\/2
\item u\+\_\+\{N-\/2\}=(1+C\+\_\+\{N-\/3\})/\+(sqrt3\texorpdfstring{$\ast$}{*}\+C\+\_\+\{N-\/2\}) (which is (\texorpdfstring{$\ast$}{*}))
\end{DoxyItemize}
