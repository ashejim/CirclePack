\doxysubsection{Detailed Description}
Creates a JButton with icon (eventually in various states),. 

Creates a JButton with icon (eventually in various states), associated Cmd\+String (with name, mnemonic, tooltip, command, dropability, etc.) useful for implementing various actions.

For \doxylink{classmytools_1_1_my_tool}{My\+Tool}\textquotesingle{}s in tool bars, execution is one of two methods:\+
\begin{DoxyEnumerate}
\item If tool is \textquotesingle{}dropable\textquotesingle{}, then execute by drag/\+drop on a canvas.
\item Otherwise, tool should be associated with an \textquotesingle{}\doxylink{classcanvasses_1_1_active_wrapper}{Active\+Wrapper}\textquotesingle{} canvas and uses its \textquotesingle{}get\+Screen\textquotesingle{} method to find the associated \textquotesingle{}CPDrawing\textquotesingle{} to be used in \textquotesingle{}jexecute\textquotesingle{} calls.
\end{DoxyEnumerate}
