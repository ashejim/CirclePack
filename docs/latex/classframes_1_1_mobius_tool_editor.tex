\doxysubsection{Detailed Description}
\doxylink{classframes_1_1_mobius_tool_editor}{Mobius\+Tool\+Editor} is a popup frame to allow the user to create. 

\doxylink{classframes_1_1_mobius_tool_editor}{Mobius\+Tool\+Editor} is a popup frame to allow the user to create tools to add to Mobius toolbar. Each tool involves a string of commands for \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack}, an Icon, and a tooltip. These can be saved in XML format to be read in on startup (going into hashed\+Tools), but the Mobius tools are generally created only when packings are read in or modified. \begin{DoxyAuthor}{Author}
