\doxysubsection{Detailed Description}
The evaluator contains the \textquotesingle{}dictionary\textquotesingle{} and. 

The evaluator contains the \textquotesingle{}dictionary\textquotesingle{} and \textquotesingle{}values\textquotesingle{} hashmaps for evaluating functions of a complex variable. The dictionary should have only objects implementing the \textquotesingle{}word\textquotesingle{} interface so that they have a complex \textquotesingle{}evaluate\textquotesingle{} method.
