\doxysubsection{Detailed Description}
Möbius transformation of the Riemann sphere, represented as a 2 × 2 complex matrix \+[a b; c d]\+. 

A Möbius (linear fractional) transformation acts on the extended complex plane (Riemann sphere) by \begin{DoxyVerb}z ↦ (az + b) / (cz + d)        if oriented,
z ↦ (a z̄ + b) / (c z̄ + d)     if not oriented (anti-conformal).
\end{DoxyVerb}


These transformations form the group PSL(2, ℂ) under composition; accordingly this class extends {\ttfamily \doxylink{classmath_1_1group_1_1_complex_transformation}{Complex\+Transformation}} and implements the {\ttfamily \doxylink{interfacemath_1_1group_1_1_group_element}{Group\+Element}} interface (left/\+right multiplication, inverse, scaling).

\begin{DoxyParagraph}{Normalization}
Orientation-\/preserving transformations are normalized so that the determinant ad − bc = 1 and the real part of the trace a + d is non-\/negative. See {\ttfamily \doxylink{classmath_1_1_mobius_abe684d7c457a73afeb71c4ed2802b322}{normalize()}}.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Classification}
A normalized Möbius transformation is classified by its trace τ = a + d (after normalizing det = 1):\+
\begin{DoxyItemize}
\item {\bfseries{Parabolic:\+}} τ real, \texorpdfstring{$\vert$}{|}τ\texorpdfstring{$\vert$}{|} = 2 (one fixed point).
\item {\bfseries{Elliptic:\+}} τ real, \texorpdfstring{$\vert$}{|}τ\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} 2 (two fixed points, rotation-\/like).
\item {\bfseries{Hyperbolic:\+}} τ real, \texorpdfstring{$\vert$}{|}τ\texorpdfstring{$\vert$}{|} \texorpdfstring{$>$}{>} 2 (two fixed points, dilation-\/like).
\item {\bfseries{Loxodromic:\+}} τ not real (two fixed points, spiral).
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Geometry support}
Factory methods construct Möbius transformations for all three geometries supported by \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} (Euclidean, hyperbolic, spherical), including normalizations that map designated circles to canonical positions (e.\+g.~{\ttfamily \doxylink{classmath_1_1_mobius_a2510fc1fe35debdb64102465eccc1737}{NS\+\_\+mobius}}, {\ttfamily \doxylink{classmath_1_1_mobius_ad5045b65e23a8c201e248da92374152e}{auto\+\_\+ab\+AB}}, {\ttfamily \doxylink{classmath_1_1_mobius_a89d8feb56adeeba2730a2c7621d06231}{standard\+\_\+mob}}).
\end{DoxyParagraph}
