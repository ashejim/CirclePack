\doxysubsection{Detailed Description}
Utility routines for working with \textquotesingle{}Path2D. 

Utility routines for working with \textquotesingle{}Path2\+D.\+Double\textquotesingle{} objects:\+ finding length, max distance from the origin, output for postscript, scaling, etc. Creating closed circle paths,

These are not so sophisticated. A key is \textquotesingle{}get\+Path\+Iterator\textquotesingle{}. If we get a \textquotesingle{}Flattening\+Path\+Iterator\textquotesingle{}, it gives lists of points to give a polygonal path approximating the path; we will just guess at a \textquotesingle{}flatness\textquotesingle{} parameter to bound how far it drifts from the actual path.
