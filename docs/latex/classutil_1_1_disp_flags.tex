\doxysubsection{Detailed Description}
Commands for displaying objects in \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} canvasses:\+ encode. 

Commands for displaying objects in \doxylink{classall_mains_1_1_circle_pack}{Circle\+Pack} canvasses:\+ encode draw, fill, thickness, and possibly depth data in compact form.

Colors=complicated:\+ at most one color can be set, color\+Is\+Set=true. E.\+g., \textquotesingle{}c25\textquotesingle{} is taken as a color code for fill. However, \textquotesingle{}cc25\textquotesingle{} would do both --- the \textquotesingle{}c\textquotesingle{} without a trailing number means to draw with colored bdry, col\+Border=true. If color is called for but color\+Is\+Set=false, then color is set elsewhere -- e.\+g. its own \textquotesingle{}color\textquotesingle{} data.
