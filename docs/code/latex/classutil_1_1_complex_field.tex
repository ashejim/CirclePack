\doxyref{Complex\+Field}{p.}{classutil_1_1_complex_field} is a panel-\/contained text field for displaying complex. 

\doxyref{Complex\+Field}{p.}{classutil_1_1_complex_field} is a panel-\/contained text field for displaying complex values and optionally includes a text label title. It is intended as a re-\/sizable replacement for the {\ttfamily \doxyref{z\+Num\+Field}{p.}{classutil_1_1z_num_field}} class.
