\doxysubsection{Detailed Description}
Our script display will be a Scroll\+Box containing a stack of \doxyref{Stack\+Box}{p.}{classscript_1_1_stack_box}\textquotesingle{}s;. 

Our script display will be a Scroll\+Box containing a stack of \doxyref{Stack\+Box}{p.}{classscript_1_1_stack_box}\textquotesingle{}s; certain \doxyref{Stack\+Box}{p.}{classscript_1_1_stack_box} can contain more \doxyref{Stack\+Box}{p.}{classscript_1_1_stack_box}\textquotesingle{}s. \doxyref{Stack\+Box}{p.}{classscript_1_1_stack_box}\textquotesingle{}s are panels, with layout depending on type and edit/\+display status. Subclasses are defined for CPscript, CPdata, Section, cmd, file, and text (often implicit) nodes; leaf nodes are cmd, file, and text and don\textquotesingle{}t have children. Currently (8/\+10) am trying to incorporate \textquotesingle{}\doxyref{Line\+SBox}{p.}{classscript_1_1_line_s_box}\textquotesingle{}s, which would simply consolidate strings of successive iconified \textquotesingle{}\doxyref{Cmd\+SBox}{p.}{classscript_1_1_cmd_s_box}\textquotesingle{}s; this is not yet completed.

If mode is NEW, an appropriate \doxyref{CPTree\+Node}{p.}{classscript_1_1_c_p_tree_node} must be created as first thing.
