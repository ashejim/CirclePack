\doxysubsection{Detailed Description}
A point on the Riemann sphere with both "{}real"{} and "{}apparent"{} coordinates. 

Extends {\ttfamily \doxyref{Complex}{p.}{classcomplex_1_1_complex}} to represent a point on the unit sphere that carries two coordinate systems simultaneously:\+
\begin{DoxyItemize}
\item {\bfseries{Real}} spherical coordinates (theta, phi) — the mathematical position on the sphere.
\item {\bfseries{Apparent}} coordinates — the screen-\/projected position after applying a {\ttfamily \doxyref{Matrix3D}{p.}{classmath_1_1_matrix3_d}} orthogonal transformation that encodes the current viewing orientation.
\end{DoxyItemize}

The inherited (x, y) fields hold the {\itshape apparent} (theta, phi) after the viewing transformation is applied in the constructor.

\begin{DoxyNote}{Note}
The apparent-\/sphere coordinates are ephemeral; they must be recomputed whenever the viewing orientation changes. The {\ttfamily \doxyref{on\+Front}{p.}{}} flag indicates whether the point is on the hemisphere facing the viewer.
\end{DoxyNote}

\textbf{ Complex} complex.\+Sph\+Point.\+matrix3\+Dz (\begin{DoxyParamCaption}\item[{boolean}]{toA}{, }\item[{\textbf{ Matrix3D}}]{trans}{, }\item[{\textbf{ Complex}}]{sph\+\_\+pt}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}



Apply an orthogonal transformation to convert between real and apparent spherical coordinates. 

When {\ttfamily to\+A == true}, the transformation maps a real (theta, phi) point to its apparent-\/sphere (theta, phi) using the orthogonal matrix {\ttfamily trans}. When {\ttfamily to\+A == false}, the inverse (transpose) is applied, mapping apparent → real coordinates.

The algorithm proceeds in three steps:\+
\begin{DoxyEnumerate}
\item Convert (theta, phi) → Cartesian (x, y, z) on the unit sphere.
\item Multiply by {\ttfamily trans} (or its transpose).
\item Convert the resulting (x, y, z) back to (theta, phi).
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
This method may no longer be called directly; see {\ttfamily to\+Apparent\+Sph} and {\ttfamily to\+Real\+Sph} in \doxyref{Sph\+View.\+java}{p.}{_sph_view_8java}.
\end{DoxyNote}
