OBE:\+ As of 3/\+2022 the \doxyref{JNI}{p.}{namespace_j_n_i} calls to C code have been deleted. This container might be used in the future.

Data container for linear system sparse matrix calls. The native code, such as C++, is given access to this when asked to solving a linear system via UMFPACK. The sizes and the arrays \textquotesingle{}Ap\textquotesingle{} and \textquotesingle{}Ai\textquotesingle{} need to be set just once. Entries \textquotesingle{}Aentries\textquotesingle{}, \textquotesingle{}rhsX\textquotesingle{}, and \textquotesingle{}rhsY\textquotesingle{} are set on each iteration. GOPacker uses the solutions \textquotesingle{}Zx\textquotesingle{} \textquotesingle{}Zy\textquotesingle{} to set new entries for next iteration.
