\doxysubsection{Detailed Description}
Red-\/chain edge:\+ tracks boundary and side-\/pairing structure in the DCEL. 

After processing a dcel (see \textquotesingle{}Comb\+DCEL.\+redcookie\textquotesingle{}), there is (except in the case of a topological sphere) a \textquotesingle{}red\+Chain\textquotesingle{}, a closed linked list of \textquotesingle{}\doxyref{Red\+Edge}{p.}{classcombinatorics_1_1komplex_1_1_red_edge}\textquotesingle{}s surrounding a simply connected patch of interior faces. In non-\/simply connected cases, segments of red edges may be identified. Outside the red\+Chain are one or more ideal faces. NOTE:\+ the dcel structure still lies with the \textquotesingle{}\doxyref{Half\+Edge}{p.}{classcombinatorics_1_1komplex_1_1_half_edge}\textquotesingle{}s via \textquotesingle{}my\+Edge\textquotesingle{}. This is not persistent data; these edges change, e.\+g., when we change the DCEL. (?? 6/\+2025 ??) TODO:\+ do we need to save data when changing the DCEL? \begin{DoxyAuthor}{Author}
