\doxysubsection{Detailed Description}
For subdivision manipulations. 

For subdivision manipulations. We follow Bill Floyd here:\+ \textquotesingle{}target\+Type\textquotesingle{} starts at 4 (because 0-\/3 are standard \textquotesingle{}end\textquotesingle{} types, used to form rectangles in the original work of Cannon, Floyd, and Parry).

However, we will (someday) introduce "{}generic"{} rules as well, and for these, \textquotesingle{}target\+Type\textquotesingle{} can be anything.

The number of vertices is \textquotesingle{}edge\+Count\textquotesingle{}, number of edges. Edges are counterclockwise; the first starts with the so-\/called \textquotesingle{}principal\textquotesingle{} corner, as used in our tiling context.
