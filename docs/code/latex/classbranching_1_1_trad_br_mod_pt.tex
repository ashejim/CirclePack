\doxysubsection{Detailed Description}
This is a traditional branch point, extra angle assigned to an interior circle. 

This is a traditional branch point, extra angle assigned to an interior circle. It is defined in this convoluted way to make processing fit the mold of more general branch types.

Changes made to \textquotesingle{}pack\+Data\textquotesingle{}:\+
\begin{DoxyItemize}
\item add to \textquotesingle{}poison\+Edges\textquotesingle{} via Edge\+Link \textquotesingle{}parent\+Poison\textquotesingle{}.
\item Aim of \textquotesingle{}my\+Index\textquotesingle{} set to -\/1 (so parent doesn\textquotesingle{}t repack)
\item send radius of \textquotesingle{}my\+Index\textquotesingle{} to parent after repack.
\end{DoxyItemize}
