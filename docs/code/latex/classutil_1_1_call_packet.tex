\doxysubsection{Detailed Description}
Utility class to pass back data generated by user. 

Utility class to pass back data generated by user command execution. Started 6/\+2015, commands will be transitioned as necessary to return \textquotesingle{}\doxyref{Call\+Packet}{p.}{classutil_1_1_call_packet}\textquotesingle{}s so the user may have programmatic access to the results.
