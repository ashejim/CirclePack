Doing more with barycentric coords in connection with 3D printing work,. 

Doing more with barycentric coords in connection with 3D printing work, so I\textquotesingle{}ve created this package (2/\+2020). Plan to migrate some other things here, maybe make a superclass. For now, I need something to hold face, barycentric coords, and some data.
