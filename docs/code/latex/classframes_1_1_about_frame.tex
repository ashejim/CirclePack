\doxysubsection{Detailed Description}
The \textquotesingle{}\doxyref{About\+Frame}{p.}{classframes_1_1_about_frame}\textquotesingle{} contains text on the left and. 

The \textquotesingle{}\doxyref{About\+Frame}{p.}{classframes_1_1_about_frame}\textquotesingle{} contains text on the left and an image on the right; it automatically pops up when loading a script that is NOT in \textquotesingle{}advanced\textquotesingle{} mode. It also pops up with the "{}\+About"{} button on the control panel or with \textquotesingle{}open about\textquotesingle{}.

The text area show "{}\+About:\+ $<$script title$>$"{}. If the script has a "{}description"{}, then that is shown. (This can give new users a place to start with the GUI.) If there is no description, then this is shown

"{}\+Edit script title to add an "{}About"{} \textquotesingle{}description\textquotesingle{}"{}

If the script contains an \textquotesingle{}About\+Image.$\ast$\textquotesingle{}, then that image is shown on the right; otherwise, the \doxyref{Circle\+Pack}{p.}{classall_mains_1_1_circle_pack} owl image is shown
