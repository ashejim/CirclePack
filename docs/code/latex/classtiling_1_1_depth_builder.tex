\doxysubsection{Detailed Description}
This class is for building subdivision tilings. 

This class is for building subdivision tilings to specified depths. It is a separate class because of the complications involved, particularly the need for recursion. In order to debug, subtasks have been split off into their own routines.

\textbf{ Pack\+Data} tiling.\+Depth\+Builder.\+build2\+Depth (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



If \textquotesingle{}t\+Data\textquotesingle{} has \textquotesingle{}sub\+Rules\textquotesingle{} and appropriate \textquotesingle{}tile\+Type\textquotesingle{}s, then build the canonical circle packing for \textquotesingle{}t\+Data\textquotesingle{} recursively to the given \textquotesingle{}depth\textquotesingle{}, attaching \textquotesingle{}t\+Data\textquotesingle{} as its \textquotesingle{}tile\+Data\textquotesingle{}. Note that depth=0 is just the canonical tiling for \textquotesingle{}t\+Data\textquotesingle{}. 

This builds a new \doxyref{Pack\+Data}{p.}{classpacking_1_1_pack_data} and updates and attaches \textquotesingle{}t\+Data\textquotesingle{} based on \textquotesingle{}t\+Data\textquotesingle{}s initial \doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data} structure. \textquotesingle{}t\+Data\textquotesingle{} must have the tiles, with consistent \textquotesingle{}tile.\+vert\textquotesingle{}s, and must have \textquotesingle{}tile\+Flower\textquotesingle{}s. (Note, eg, that for subdivision rules, may have to run \textquotesingle{}Subdivsion\+Rules.\+get\+Rules\+TD\textquotesingle{} to set consistent vertex indices.) If the tiles have non-\/null \textquotesingle{}my\+Tile\+Data\textquotesingle{}, then they themselves represent tilings (as when we build from subdivision rules).

Therefore, this is a recursive construction:\+ At a given level, a tiling is created one tile at a time with its associated packing; this is pasted onto a growing global packing (for this level) and the \doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data} tiles are updated, with \textquotesingle{}augmented\textquotesingle{} vertices from the local packing; as that is absorbed into the growing global packing, we update \textquotesingle{}vert\textquotesingle{} and \textquotesingle{}aug\+Vert\textquotesingle{} indices to the parent packing and then discard the local packing to save space (but the local \textquotesingle{}\doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data}\textquotesingle{} is now part of the parent\textquotesingle{}s \doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data}).

We do not build a global \textquotesingle{}\doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data}\textquotesingle{} that has tiles down to the finest level. Instead, each \textquotesingle{}\doxyref{Tile}{p.}{classtiling_1_1_tile}\textquotesingle{} at level j$>$0 contains \textquotesingle{}\doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data}\textquotesingle{} for j-\/1 level tiles.

TODO:\+ build a "{}collapse"{} method that builds the tiling down to the finest level for purposes of saving it.
