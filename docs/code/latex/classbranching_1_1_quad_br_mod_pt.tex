\doxysubsection{Detailed Description}
The "{}quad"{} branching was intended to branching that occurs. 

The "{}quad"{} branching was intended to branching that occurs precisely on an edge between two faces. The numerics are sensitive near tangency points where two circles and two interstices come together, so James Ashe had an idea on a special type of generalized branch point.
