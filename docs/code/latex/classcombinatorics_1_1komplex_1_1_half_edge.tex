\doxysubsection{Detailed Description}
DCEL half-\/edge:\+ the fundamental directed-\/edge element of the combinatorial structure. 

In a doubly-\/connected edge list (DCEL), every undirected edge is represented by a pair of oppositely oriented half-\/edges linked via {\ttfamily \doxyref{twin}{p.}{classcombinatorics_1_1komplex_1_1_half_edge_ab37a4ceb1fc3f962c4ab14c9427af714}}. The face to the {\itshape left} of each half-\/edge is stored in {\ttfamily \doxyref{face}{p.}{classcombinatorics_1_1komplex_1_1_half_edge_a4155eca1928f96d1786bc1836f286e41}}. Half-\/edges around a face are linked by {\ttfamily \doxyref{next}{p.}{classcombinatorics_1_1komplex_1_1_half_edge_a5767a3d5e5ec349fd34bf4ec753ce0a7}} and {\ttfamily \doxyref{prev}{p.}{classcombinatorics_1_1komplex_1_1_half_edge_ab543b5039f5c326862301fa695429291}} in counter-\/clockwise order.

Each half-\/edge also carries per-\/edge packing data:\+
\begin{DoxyItemize}
\item {\ttfamily \doxyref{inv\+Dist}{p.}{}} — inversive distance (default 1.\+0 = tangency).
\item {\ttfamily \doxyref{schwarzian}{p.}{}} — discrete Schwarzian derivative.
\item {\ttfamily \doxyref{color}{p.}{}} — display color.
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
Half-\/edges are {\itshape not persistent} across combinatorial modifications (flips, refinements, etc.). Code that caches pointers to half-\/edges must revalidate after any structural change.
\end{DoxyWarning}
