\doxysubsection{Detailed Description}
This manager is created on startup for managing directory paths,. 

This manager is created on startup for managing directory paths, opening Buffered\+Reader/\+\+Writers, etc. Reading/\+writing routines themselves are generally in appropriate classes.

Have some static file routines, such as copy\+File.

In searching for resources (e.\+g., \textquotesingle{}mytool\textquotesingle{} files), we generally have to search the home directory, since current\+Directory via the jar may be variable.
