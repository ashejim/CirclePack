Store and maintain information on a chain (segment) of. 

Store and maintain information on a chain (segment) of edges. (Used, e.\+g., in \textquotesingle{}\doxyref{Rational\+Map.\+java}{p.}{_rational_map_8java}\textquotesingle{}.) Just need the first and last vertices, which should be boundary vertices, defining the cclw edge segment. Keep number of edges for compatibility checks.
