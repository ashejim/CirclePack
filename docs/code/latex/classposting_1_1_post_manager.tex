\doxysubsection{Detailed Description}
/\+$\ast$$\ast$ For opening/\+augmenting/\+closing postscript output files. For opening/\+augmenting/\+closing postscript output files. Status of \textquotesingle{}fp\textquotesingle{} is important for opening/\+closing/\+adding decisions, etc. This manager persists, but \textquotesingle{}fp\textquotesingle{} changes. 

This is all rather complicated since stuff can be added to the postscript file in increments, pages can be defined, material inserted, etc. The final file is created only when it is about to be closed:\+ then we open the file and


\begin{DoxyEnumerate}
\item add prefix material (where, eg. \textquotesingle{}Bounding\+Box\textquotesingle{} may be adjusted for text at end); see \textquotesingle{}initialize\textquotesingle{}
\item copy in the main body, temporary in file, \textquotesingle{}body\+File\textquotesingle{}; this is where the \textquotesingle{}fp\textquotesingle{} file pointer is set and where \textquotesingle{}post\+Factory\textquotesingle{} writes stuff.
\item copy in trailer stuff accummulated in \textquotesingle{}tail\textquotesingle{} file.
\item close with any text accummulated in \textquotesingle{}text\+Buffer\textquotesingle{}.
\end{DoxyEnumerate}

Also, there are questions of interaction with user and "{}\+Post\+Script"{} tab, errors to catch, filename, where to store.
