\doxysubsection{Detailed Description}
Hamilton quaternion q = q₁ + q₂i + q₃j + q₄k, usable as a group element for representing rotations in 3-\/space. 

Quaternions form a non-\/commutative group under multiplication and can also be added. A unit quaternion q encodes a rotation in 3-\/space via the map v ↦ q v q⁻¹ (where v is a "{}pure"{} quaternion with q₁ = 0). A rotation by angle θ about a unit axis (a, b, c) corresponds to q = (cos θ/\+2, a sin θ/\+2, b sin θ/\+2, c sin θ/\+2).

\begin{DoxyNote}{Note}
Currently used sparingly in \doxyref{Circle\+Pack}{p.}{classall_mains_1_1_circle_pack}; most rotations use {\ttfamily \doxyref{Mobius}{p.}{classmath_1_1_mobius}} or {\ttfamily \doxyref{Matrix3D}{p.}{classmath_1_1_matrix3_d}} instead.
\end{DoxyNote}
