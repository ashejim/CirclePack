\doxysubsection{Detailed Description}
This \textquotesingle{}Pack\+Extender\textquotesingle{} was initiated for preliminary experiments with. 

This \textquotesingle{}Pack\+Extender\textquotesingle{} was initiated for preliminary experiments with the expectation that it could be developed incrementally.

Basic notion of feedback is that boundary radii of a euclidean packing can be adjusted based on other radii in the packing (eventually, perhaps another packing).

I first experimented with feedback from the inner "{}rectangle"{} patch of the dodecahedral tiling to the outer boundary to see if the overall global shape would stabilize. \begin{DoxyAuthor}{Author}
