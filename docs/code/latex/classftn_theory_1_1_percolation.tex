\doxysubsection{Detailed Description}
\textquotesingle{}\doxyref{Percolation}{p.}{classftn_theory_1_1_percolation}\textquotesingle{} is at least a start at experiments in this 

\textquotesingle{}\doxyref{Percolation}{p.}{classftn_theory_1_1_percolation}\textquotesingle{} is at least a start at experiments in this topic. A first target is Cardy\textquotesingle{}s Theorem, though the route via a \textquotesingle{}hexamania\textquotesingle{} type approach is something I still don\textquotesingle{}t understand.

In any case, the setting will be a simply connected complex K on whose boundary we mark four segments, two blue and the intervening two red. We investigate open percolation paths from blue to blue, and red to red. We keep track of the state of vertices via their \textquotesingle{}color\textquotesingle{}:\+ blue=1 and blue=2 for the blue bdry arc and its opposite, red=198 and red=199 for the other two. We randomly choose unmarked (color=0) interior vertices, do a random walk to see which state it encounters first, then mark it accordingly as follows:\+

vertices that are "{}infected"{}:\+ color 1:\+ path connected by 1\textquotesingle{}s to the blue=1 edge color 2:\+ path connected by 2\textquotesingle{}s to the blue=2 edge ~\newline
 color 199:\+ path connected by 199\textquotesingle{}s to the red=199 edge color 198:\+ path connected by 198\textquotesingle{}s to the red=198 edge

vertices that are "{}marked"{} (but not yet infected):\+ color 244:\+ random walk first encounters one of blue arcs color 243:\+ random walk first encounters one of red arcs

Suppose a vertex v is just marked. You look for a petal p that is infected with the corresponding color (1, 2, 199, or 198). Then you recursively start with p to infect neighbors:\+ if petal u has corresponding mark, then u becomes infected with same color as p. (Thus, v gets infected, e.\+g.).

You follow infection recursively, thus building up connected components of vertices infected with the same color. If a newly infected vert has a petal which is infected by the \textquotesingle{}opposite\textquotesingle{} mark, then \textquotesingle{}completed\textquotesingle{} is set to show that a connecting path has been completed. E.\+g. if infected is just set to 1 and a petal is 2, then set completed to 1 (and vice verse); while if infected is just set to 198 and petal is 199, then set completed to 198. vert ~\newline
