\doxysubsection{Detailed Description}
A parent tile edge is broken into some number of. 

A parent tile edge is broken into some number of sub-\/edges. These are listed cclw. For example if the edge is broken into 3 subedges we have data such as "{}1   1 2   1 1   0 3"{} which indicates that as you go cclw along edge 1, you encounter edge 2 of subtile 1, then edge 1 of subtile 1, and finally edge 3 of subtile 0. CAUTION:\+ subtile indices in the original rules file have been incremented to start at 1.

Store:\+ subedges are indexed from 0.
\begin{DoxyItemize}
\item \textquotesingle{}tileedge[j][0]\textquotesingle{} = tile index of \textquotesingle{}j\textquotesingle{}\+\_\+th child tile t counterclockwise along this edge.
\item \textquotesingle{}tileedge[j][1]\textquotesingle{} = the index in \textquotesingle{}t.\+vert\textquotesingle{} of the first leading vertex in the side of t which lies along this parent edge.
\end{DoxyItemize}
