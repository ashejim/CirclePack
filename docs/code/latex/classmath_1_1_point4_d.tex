\doxysubsection{Detailed Description}
A point in 4-\/D Lorentz (Minkowski) space (t, x, y, z). 

In the Lorentz model of hyperbolic geometry, circles on the unit sphere are represented by points satisfying the Lorentz condition t² − x² − y² − z² = −1. The corresponding spherical circle is \{(a,b,c) :\+ a²+b²+c²=1, ax+by+cz = t\}.

The Lorentz inner product is ⟨u,v⟩ = u\+\_\+t·v\+\_\+t − u\+\_\+x·v\+\_\+x − u\+\_\+y·v\+\_\+y − u\+\_\+z·v\+\_\+z (signature (1,3)), so the "{}norm"{} can be positive, negative, or zero.
