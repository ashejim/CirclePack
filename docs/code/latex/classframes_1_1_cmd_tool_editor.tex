\doxysubsection{Detailed Description}
Cmd\+Tool\+Frame is a popup frame to allow the user to create. 

Cmd\+Tool\+Frame is a popup frame to allow the user to create tools to add to My\+Tools. Each tool involves a string of commands for \doxyref{Circle\+Pack}{p.}{classall_mains_1_1_circle_pack}, an icon, and a tooltip.
