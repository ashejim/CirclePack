\doxysubsection{Detailed Description}
Lightweight circle data container for transferring center/\+radius between methods. 

Basic, simple circle, mainly for transferring data in/\+out of methods. Note:\+ geometry is NOT specified. May use \textquotesingle{}flag\textquotesingle{} to codes for interpreting output/\+input. Normally, flag=-\/1 on error, flag=1 on success, flag=0 neutral. Other options may be agreed in particular calls. line\+Flag:\+ false:\+ regular circles (rad$<$0 means disc is the outside) true:\+ center is unit normal vector towards interior times \doxyref{CPBase.\+FAUX\+\_\+\+RAD}{p.}{classall_mains_1_1_c_p_base_ac8b0dc617a051fc497c9d5d002829c45}, while -\/rad is FAUX\+\_\+\+RAD plus signed distance from origin (so if rad$>$FAUX\+\_\+\+RAD, then origin is "{}interior"{} and line is shifted by vector -\/rad$\ast$unit normal.) NOTE:\+ \textquotesingle{}lineflag\textquotesingle{} is only used in eucl setting; it is NOT set in spherical geometry for circles through infinity.
