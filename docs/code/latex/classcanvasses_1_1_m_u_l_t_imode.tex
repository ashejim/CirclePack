\doxysubsection{Detailed Description}
===== active canvas cursor mode ==== 

===== active canvas cursor mode ==== This is for using pencil icon to draw paths, but allowing for multiple closed components; see \doxyref{PATHmode}{p.}{classcanvasses_1_1_p_a_t_hmode} for single components. Note:\+ when a path component is closed, the full path is put into Global\+Path and polygonal\+Path and my\+Path are set to null. When forming a new component, you can tell if there was a preexisting path by checking polygonal\+Path!=null.
