\doxysubsection{Detailed Description}
Streamline generator for curvature-\/flow lines on circle packings. 

A class for creating streamlines, a la curvature flow lines, as Chuck Collins did using matlab in joint paper with me and Toby Driscoll.

Steamlines are associated with a packing and are represented in \textquotesingle{}Bary\+Link\textquotesingle{}s of of \textquotesingle{}\doxyref{Bary\+Packet}{p.}{classbary_stuff_1_1_bary_packet}\textquotesingle{}s, hence in barycentric coordinates.

The \textquotesingle{}base\+Pack\textquotesingle{} is euclidean, contains the combinatorics and vertex locations. The base\+Pack typically provides a way to plot the streamlines, since the packing used to create them may not have a convenient layout.
