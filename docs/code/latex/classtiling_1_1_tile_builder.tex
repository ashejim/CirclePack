\doxysubsection{Detailed Description}
\doxyref{Tile\+Builder}{p.}{classtiling_1_1_tile_builder} is intended for building \textquotesingle{}\doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data}\textquotesingle{} structures. 

\doxyref{Tile\+Builder}{p.}{classtiling_1_1_tile_builder} is intended for building \textquotesingle{}\doxyref{Tile\+Data}{p.}{classtiling_1_1_tile_data}\textquotesingle{} structures and their packings.

TODO:\+ trying to fix multiply connected situations

Basic method here is build tile-\/by-\/tile so we can handle situations that come up when applying subdivision rules, building dessins, fusion tilings, etc. Namely, we need to handle unigons, digons, and various self-\/pastings.

For recursive use, we need to allow \textquotesingle{}orig\+TD\textquotesingle{} to have some "{}null"{} tiles:\+ E.\+g. \textquotesingle{}orig\+TD\textquotesingle{} may be a connected subset of tiles from a larger tiling, but it keeps the original tile indices and inherits original \textquotesingle{}my\+Tiles\textquotesingle{} length.

Idea:\+ start with root\+Tile and keep adding tiles. There are many complications. E.\+g., a loop is an edge with same vertex at its ends, and to paste along a loop in \textquotesingle{}master\+Pack\textquotesingle{} requires special \textquotesingle{}adjoin\textquotesingle{} method:\+ do all but one of the sub-\/edge pastings with usual adjoin, fix up the packing, then do one last self-\/adjoin to close the last edge. However, this may require recursive calls so the stuff being adjoined, if not just a unigon, has already been consolidated in a simply connected packing.

Issues:\+
\begin{DoxyItemize}
\item tiles may attach along edges with multiple vertices
\item tiles may attach to themselves (more than once, even) along edges or at vertices
\item two tiles may be attached along more than one, perhaps non-\/contiguous edges.
\item tiles may be unigons or digons
\item tiles must have \textquotesingle{}tileflower\textquotesingle{} data:\+
\item t=tile.\+tileflower[i][0] is the index of the tile across edge i (if t=0, this is a bdry edge) and tile.\+tileflower[i][1] is the index of the shared edge in t.
\end{DoxyItemize}

Build associated full barycentric complex for orig\+TD:\+ that is, every n-\/tile is barycentrically subdivided to form 2n triangles, and then each of these is barycentrically subdivided. (This is required, e.\+g., to handle, digons, slits, and some self-\/pastings. We have no consistent way to handle uniqons, so things will fail there.) Vertices get \textquotesingle{}mark\textquotesingle{}s \{1,2,3\} if they are, resp, bary\+Vert, original corner, edge barycenter (for Belyi map values \{infty, 0,1\}, resp). Also, barycenter of each of the 2n barycentrically divided faces is degree 6 and is marked -\/1. 

Strategy:\+
\begin{DoxyItemize}
\item we work purely from \textquotesingle{}tile\+Flower\textquotesingle{} data, so we discard and reconstitute all vertex indices using \textquotesingle{}tileflowers2verts\textquotesingle{}.
\end{DoxyItemize}

TODO:\+ See if this first step can be avoided and we can restore the original indices for the vertices of the original tiles.


\begin{DoxyItemize}
\item We create a packing for each tile as needed:\+ this is a barycentrically subdivided 2n-\/gon, recording original corner indices from tile (tile\+\_\+v,pack\+\_\+v) in its \textquotesingle{}\doxyref{Vertex\+Map}{p.}{classlist_manip_1_1_vertex_map}\textquotesingle{}. Note that each tile side has 4 edges.
\item the first tile is chosen as root\+Tile; we may need to make some choice so we have an easy base.
\item Each tile, including the first, is processed for slit-\/type self-\/pastings, surrounded unigons, etc. (E.\+g. suppose an n-\/gon shares one edge with a unigon; then we create and paste in that unigon.) (we\textquotesingle{}ll see what other circumstances show up)
\item We have the growing \textquotesingle{}master\+Pack\textquotesingle{}. We search through unpasted, non-\/bdry tile edges to find one which can be unambiguously pasted.
\item We create the packing for a new tile (processed as above), paste it in, then look for any self-\/pastings this may lead to for \textquotesingle{}master\+Pack\textquotesingle{} itself.
\item We paste, adjust the \textquotesingle{}\doxyref{Vertex\+Map}{p.}{classlist_manip_1_1_vertex_map}\textquotesingle{}, and iterate until all tiles are included and all non-\/bdry tile edges are pasted.
\item We build \textquotesingle{}dual\textquotesingle{}, \textquotesingle{}quad\textquotesingle{}, and \textquotesingle{}wg\textquotesingle{} tilings.
\end{DoxyItemize}

\begin{DoxyItemize}
\item ans[0] = index in \textquotesingle{}t1.\+vert\textquotesingle{} where pasting starts
\item ans[1] = number of clockwise tile edges to paste
\item ans[2] = index in \textquotesingle{}t2.\+vert\textquotesingle{} where pasting starts
\item ans[3] = number of counterclockwise tile edges to paste. Both t1 and t2 must have \textquotesingle{}tile\+Flower\textquotesingle{}s. (This routine follows the orientation conventions of \textquotesingle{}adjoin\textquotesingle{} when adjoining t2 to t1.)
\end{DoxyItemize}
