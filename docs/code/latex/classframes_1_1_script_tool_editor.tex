\doxyref{Script\+Tool\+Editor}{p.}{classframes_1_1_script_tool_editor} is a popup frame to allow the user to create tools to add to the script. Each tool involves a string of commands for CPack, an Icon, and a tooltip. These can be saved in XML format and are then read in on startup, going into hashed\+Tools. \begin{DoxyAuthor}{Author}
