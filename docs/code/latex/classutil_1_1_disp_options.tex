\doxyref{Disp\+Options}{p.}{classutil_1_1_disp_options} simply holds the default display options for a given circle. 

\doxyref{Disp\+Options}{p.}{classutil_1_1_disp_options} simply holds the default display options for a given circle packing and supplies it to \textquotesingle{}disp\textquotesingle{} commands when specific flags are lacking. (\textquotesingle{}disp\textquotesingle{} and \textquotesingle{}disp -\/wr\textquotesingle{} calls in \textquotesingle{}Pack\+Control.\+naked\+Disp\textquotesingle{}).
