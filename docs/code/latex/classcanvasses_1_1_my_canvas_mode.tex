\doxysubsection{Detailed Description}
\doxyref{My\+Canvas\+Mode}{p.}{classcanvasses_1_1_my_canvas_mode} is used to associate modes/\+cursors/\+icons to. 

\doxyref{My\+Canvas\+Mode}{p.}{classcanvasses_1_1_my_canvas_mode} is used to associate modes/\+cursors/\+icons to \textquotesingle{}\doxyref{Active\+Wrapper}{p.}{classcanvasses_1_1_active_wrapper}\textquotesingle{}s. It holds the icon, \textquotesingle{}mode\+Type\textquotesingle{} of creation, (e.\+g., from \textquotesingle{}My\+Tool\textquotesingle{}), and catches mouse events. Canvas modes are kept in vectors in \textquotesingle{}\doxyref{Active\+Wrapper}{p.}{classcanvasses_1_1_active_wrapper}\textquotesingle{} and persist. Mouse events are caught in \textquotesingle{}\doxyref{Active\+Wrapper}{p.}{classcanvasses_1_1_active_wrapper}\textquotesingle{}, but processed in \textquotesingle{}Active\+Mode\textquotesingle{}, including decisions about repaints.

NOTE:\+ if created in script or my\+Tool bar, there may be an associated \textquotesingle{}My\+Tool\textquotesingle{} which appears on tool bars. \begin{DoxyAuthor}{Author}
