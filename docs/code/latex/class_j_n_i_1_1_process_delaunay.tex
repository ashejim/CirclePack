Static routines for creating Delaunay triangulations with calls to \textquotesingle{}qhull\textquotesingle{} (for the sphere) and \textquotesingle{}triangle\textquotesingle{} for regions in the plane. I\textquotesingle{}m using \textquotesingle{}Process\+Builder\textquotesingle{}.
