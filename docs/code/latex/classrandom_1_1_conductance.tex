Save adjacency matrix of packing ($<$= 10,000 verts) in form for matlab to read; the adjacency matrix has a 1 in (i,j) spot if i and j are neighboring vertices, else a 0. 

If \textquotesingle{}tm\textquotesingle{} is true, divide every row by sum of its entries and set the rows for boundary vertices to zero:\+ this yields the "{}transition"{} matrix for the "{}simple"{} (ie. equal probability) random walk on the edge-\/graph of the packing with absorbing boundary. Also add an ordered list of the boundary vertices to the matlab file.
