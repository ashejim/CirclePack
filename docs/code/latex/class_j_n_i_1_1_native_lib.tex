\doxysubsection{Detailed Description}
This code was needed for cross-\/platform access to the C/\+\+C++. 

This code was needed for cross-\/platform access to the C/\+\+C++ libraries when I was using \doxyref{JNI}{p.}{namespace_j_n_i} (Java Native Interface) calls. As of 3/\+2022 I\textquotesingle{}ve moved towards calls via \textquotesingle{}Process\+Builder\textquotesingle{} and using in/\+output files.

Various versions of the libraries were compiled in the past. This may still be necessary, but is currently beyond my abilities. At runtime the libraries appropriate to the operating system and computer should be stored in temporary directories and the system told where to find them.
