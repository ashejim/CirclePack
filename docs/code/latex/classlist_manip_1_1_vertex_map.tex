In many situations, we do repeated adjustments in DCEL and need to connect a final new index with its original from several steps back. Start with null \doxyref{Vertex\+Map}{p.}{classlist_manip_1_1_vertex_map}, and each stage compose with \doxyref{Vertex\+Map}{p.}{classlist_manip_1_1_vertex_map} from that stage. Return the composition:\+ \textquotesingle{}domain\+Map\textquotesingle{} followed by given \textquotesingle{}range\+Map\textquotesingle{}. Typically, for (b,c) in \textquotesingle{}range\+Map\textquotesingle{}, look for (a,b) in \textquotesingle{}domain\+Map\textquotesingle{}. If found, put (a,c) in output map. Some details:\+ 


\begin{DoxyItemize}
\item if (b,c) is in range\+Map and ({\itshape ,b) does not exist in domain\+Map, then (b,c) is inserted in output. (I.\+e., as though (b,b) were in domain\+Map).}
\item {\itshape may be ambiguous:\+ if (a1,b) before (a2,b) in \textquotesingle{}domain\+Map\textquotesingle{}, then (a1,c) is the result (i.\+e., the first occurrance of ($\ast$.b)).}
\item {\itshape However, if (b,c1) before (b,c2) in \textquotesingle{}range\+Map\textquotesingle{}, then ($\ast$,c2) is the outcome (i.\+e., the last occurrance of (b,})).
\item if \textquotesingle{}domain\+Map\textquotesingle{} null, return \textquotesingle{}range\+Map\textquotesingle{}
\item if \textquotesingle{}range\+Map\textquotesingle{} null, return \textquotesingle{}domain\+Map\textquotesingle{}
\item if (b,c) in \textquotesingle{}range\+Map\textquotesingle{} and b==c, then ignore. Note:\+ typically, not including (x,x) entries in \textquotesingle{}\doxyref{Vertex\+Map}{p.}{classlist_manip_1_1_vertex_map}\textquotesingle{}s
