\doxysubsection{Detailed Description}
Slider panels occur in \textquotesingle{}Pack Info\textquotesingle{} under the \textquotesingle{}Variables\textquotesingle{} tab. 

Slider panels occur in \textquotesingle{}Pack Info\textquotesingle{} under the \textquotesingle{}Variables\textquotesingle{} tab in a region below the variables table. Each displays its variable name, the slider, slider min/\+max settings, a \textquotesingle{}ftn?\textquotesingle{} checkbox (e.\+g. with z$\ast$pi to multiply values by pi), slider, current value, and optional command execution button and string.

See \textquotesingle{}Variable\+Control\+Table\+Model\textquotesingle{} in \textquotesingle{}\doxyref{Var\+Control.\+java}{p.}{_var_control_8java}\textquotesingle{}
