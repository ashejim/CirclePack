This is where individual commands are analyzed and sent to appropriate Java routines. 

Commands arriving are individual commands:\+ preprocessing leaves no \textquotesingle{};\textquotesingle{} separators:\+ no \textquotesingle{}for\textquotesingle{} or \textquotesingle{}FOR\textquotesingle{} loops:\+ no \textquotesingle{}delay\textquotesingle{}s:\+ no [n] named commands to resolve:\+ no \textquotesingle{}!!\textquotesingle{} repeats to resolve:\+ no \textquotesingle{}$\vert$pe$\vert$\textquotesingle{} \textquotesingle{}\doxyref{Pack\+Extender}{p.}{classpacking_1_1_pack_extender}\textquotesingle{} calls:\+ no \textquotesingle{}-\/p\textquotesingle{} flag (already caught):\+

This routine handles housekeeping, separating cmd and flag sequences, then catching a small number of certain commands. Note that there is a call to \textquotesingle{}pack\+Execute\textquotesingle{} for commands requiring pack status. Commands not requiring pack status, are then processed in a switch based on the command\textquotesingle{}s first letter.
