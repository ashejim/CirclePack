\doxysubsection{Detailed Description}
Static utility functions for complex arithmetic. 

This companion class to {\ttfamily \doxyref{Complex}{p.}{classcomplex_1_1_complex}} collects static two-\/argument operations (add, sub, mult, divide, pow, …) and numerical predicates (is\+Small, is\+Integer, distance) that do not belong on a single \doxyref{Complex}{p.}{classcomplex_1_1_complex} instance. It also defines the fundamental complex constants {\ttfamily \doxyref{IM}{p.}{classcomplex_1_1_math_complex_ab37d5c91eaff3ff63950df9183cdccab}}, {\ttfamily \doxyref{ZERO}{p.}{classcomplex_1_1_math_complex_af21e2b44718ff298de53f867c906be31}}, and {\ttfamily \doxyref{ID}{p.}{classcomplex_1_1_math_complex_a0aad7359133b45bfa242c2eaba9767f3}}.

\begin{DoxyNote}{Note}
The tolerance used by {\ttfamily \doxyref{is\+Small(double)}{p.}{classcomplex_1_1_math_complex_a3193cacd75377471e615352d81a88264}} and related checks is currently hard-\/coded at 1 × 10⁻⁷.
\end{DoxyNote}
