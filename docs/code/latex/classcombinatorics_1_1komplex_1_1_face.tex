\doxysubsection{Detailed Description}
Legacy face data structure (pre-\/\+DCEL); retained for compatibility. 

Structure for face information; this was designed for triangular faces, later modified for more general polygonal faces. Note that in Circle\+Pack, face indices are not persistent but created on the fly. \begin{DoxyAuthor}{Author}
