\doxysubsection{Detailed Description}
Static methods for mathematical operations in Euclidean geometry. 

Provides the Euclidean-\/specific implementations of circle-\/packing computations:\+ angle sums from radii, face layouts, tangent-\/point calculations, overlap angles, and related geometric utilities. The Euclidean metric is simply the standard flat metric on ℝ².
