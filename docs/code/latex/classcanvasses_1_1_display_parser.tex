\doxysubsection{Detailed Description}
This static code parses the vector of display flag segments. 

This static code parses the vector of display flag segments for canvasses and carries out the specified display actions.

Note:\+ the first version is the usual one, but the second allows us to separate the packing from its screen, so one can put images from another packing on an existing screen.

Preliminary clear/\+repeat and closing repaint actions are handled in the calling routine. \begin{DoxyAuthor}{Author}
