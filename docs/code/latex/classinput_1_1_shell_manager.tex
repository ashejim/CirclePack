\doxysubsection{Detailed Description}
This class manages the static command history buffer, \textquotesingle{}run\+History\textquotesingle{}. 

This class manages the static command history buffer, \textquotesingle{}run\+History\textquotesingle{} I am trying to run the \doxyref{Circle\+Pack}{p.}{classall_mains_1_1_circle_pack} GUT via a unix style \textquotesingle{}shell\textquotesingle{}. Successive user commands and resulting messages and error messages are put into the \textquotesingle{}run\+History\textquotesingle{} String\+Buffer by this manager or some other method.

Details:\+
\begin{DoxyItemize}
\item \textquotesingle{}run\+History\textquotesingle{} is in html format
\item end of the header lines is "{}$<$body$>$\textbackslash{}n"{}
\item the String\+Buffer has a max size to which text is trimmed when necessary by removing full lines at the top, after $<$body$>$~\newline
.
\item Many commands (e.\+g. drawing ops, some tools) are not recorded in the history, so it can\textquotesingle{}t duplicate a saved run.
\end{DoxyItemize}
