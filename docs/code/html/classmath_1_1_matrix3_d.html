Detailed Description

3 × 3 real matrix for rotations, rigid motions, and general linear algebra on 3-D vectors.

In CirclePack this class is used primarily for SO(3) rotation matrices that orient the "apparent sphere" (the viewer's perspective of the Riemann sphere). For an orthogonal matrix, the inverse is simply the transpose.

Coordinate-axis rotation conventions (right-handed system)
  RX =  1       0        0        (ccw in yz-plane)
        0     cos t   −sin t
        0     sin t    cos t

  RY =  cos t    0     sin t      (ccw in zx-plane)
          0      1       0
       −sin t    0     cos t

  RZ =  cos t  −sin t    0        (ccw in xy-plane)
        sin t   cos t    0
          0       0      1

double yRot,
double zRot )

Matrix determined by the product of rotations about axis of xyz coord system (treated as fixed). Each angle in radians, counterclockwise rotation as viewed from the positive end of the axis involved; first apply z-rotation, then x-rotation, then y-rotation. (I'm not sure of the connection to Euler angles.).

Parameters
xRot,rotationof (y,z) plane
yRot,rotationof (x,z) plane
zRot,rotationof (x,y) plane
Returns
Matrix3D=yRot*xRot*zRot (Note: this is different from the original methods of CirclePack, which rotated about y in the opposite direction and used order xRot*yRot*zRot.)
