Detailed Description

HoverAdapter is the internal class of FluidHoverPanel that controls the appearance of the tooltip panel. It is a MouseAdapter that gets attached to the parent component when FluidHoverPanel is instantiated, and reacts to MouseEvents received by the parent component.
