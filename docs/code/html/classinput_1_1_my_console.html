Detailed Description

Create consoles which handle user entry of CirclePack commands:

Create consoles which handle user entry of CirclePack commands: (1) Package commands to send to 'commandStrParser', (2) Maintain a "Command:" line for user entry of commands, with command completion, emacs keys. (3) If desired, maintain the one-line success count/message line below the command window. (4) if desired, call 'LockingShell' hover (developed by Alex Fawkes)
