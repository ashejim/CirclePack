ImageIcon images.CPIcon.embellishIcon ( ImageIcon startImg,

boolean xy_tear,
boolean hot_arrow,

boolean menu_pop )
static

This returns modified image with embellishments:

  1. letter for mnemonic
  2. Abbreviation for 'PackExtender's (not yet implemented)
  3. blue drop for dropable with #XY (or #xy)
  4. hotpoint for cursor, lower left
  5. hand for cursor, lower right
  6. popup menu indicator Note: icon can change size
    Parameters
    startImg,ImageIcon
    key,String,forkey associated to tool
    booleanxy_tear (blue teardrop for dropable at XY point) hot_arrow (arrow point (lower left) for mouse) handy (hand at lower right, precedence over tear) menu_pop
