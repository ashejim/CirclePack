Detailed Description

Store and maintain information on a chain (segment) of.

Store and maintain information on a chain (segment) of edges. (Used, e.g., in 'RationalMap.java'.) Just need the first and last vertices, which should be boundary vertices, defining the cclw edge segment. Keep number of edges for compatibility checks. Have routines to use 'VertexMap' to modify the segment indices when vertices get renumbered.
