Detailed Description

Our script display will be a ScrollBox containing a stack of StackBox's;.

Our script display will be a ScrollBox containing a stack of StackBox's; certain StackBox can contain more StackBox's. StackBox's are panels, with layout depending on type and edit/display status. Subclasses are defined for CPscript, CPdata, Section, cmd, file, and text (often implicit) nodes; leaf nodes are cmd, file, and text and don't have children. Currently (8/10) am trying to incorporate 'LineSBox's, which would simply consolidate strings of successive iconified 'CmdSBox's; this is not yet completed.

If mode is NEW, an appropriate CPTreeNode must be created as first thing.
