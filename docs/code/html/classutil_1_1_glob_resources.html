Convert a glob pattern into a regular expression.

NOTE: The only special characters recognized are '*' for "match any sequence of characters" and '?' for "match any one character. Also, these characters cannot be escaped in the pattern. This keeps the code below much simpler, and it seems unlikely that such escaping would be useful.
