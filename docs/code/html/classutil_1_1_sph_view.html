Detailed Description

For maintaining viewpoint for spherical packings.

For maintaining viewpoint for spherical packings. Spherical centers are given as complex numbers, but really they are (theta, phi) pairs; radii are angles from 0 to Pi. (Everything is done in radian measure.)
