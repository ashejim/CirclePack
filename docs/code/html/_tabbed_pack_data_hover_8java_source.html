141 * Some notes on using BoxLayout:

144 * hierarchy of panels and ultimately components in terms
145 * of rows and columns. In this case, the ultimate parent
146 * is this instance, which is a JPanel. It contains two
147 * components as rows - the tabbed pane containing the
148 * bulk of the GUI, and the update button.

150 * The first tab of the tabbed pane is the combinatoric
151 * information panel. It contains three rows - the vertex

155 * columns - the degree field, the flower field, the angle

689 public void updateVertex(PackData p, boolean useActiveVertex) {
690 // If packData is null or empty just return.
691 if (p == null || !p.status) // p.packDCEL.vertices[2].center
692 return;
693
694 // Update for the current active or chosen vertex,
695 // depending on the call signature.
696 int v;
697 try {
698 if (useActiveVertex)
699 v = p.activeNode;
700 else {
