Detailed Description

For MyTool drag/drop operation.

For MyTool drag/drop operation. This is the listener for the targets, which are currently the active canvas, the three smaller canvasses, and the canvasses of PairFrame. WhichPackFlag true means that the packing number must be determined from the target panel, so we have to search for which panel.
