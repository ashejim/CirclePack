Detailed Description

The 'LineSBox' is an ephemeral abstract node for consolidating.

The 'LineSBox' is an ephemeral abstract node for consolidating a contiguous sequence of two or more unopened 'CmdSBox's or 'FileSBox's. They are created only in 'CPTreeNode.consolidateNodes'.
