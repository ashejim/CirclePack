Detailed Description

This class can extract an image from a valid and compatible XML resource.

This class can extract an image from a valid and compatible XML resource. Valid but incompatible XML resources may be passed without error. In this implementation, compatibility means that the XML resource contains an element whose content is a preview image encoded as a valid base 64 string and whose tag is as defined in PreviewImageSearchHandler. Instances should be initialized with the target URL, after which the image may be requested.
