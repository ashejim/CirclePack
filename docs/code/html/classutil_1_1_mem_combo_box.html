void util.MemComboBox.add2List ( String itemname,
boolean loadOK )

Add 'itemname' to stored list held for this combo box; name moves (or is added) to stored list so it comes up first next time CirclePack is run. Don't move it into the model's element list since that triggers reloading actions. Just fix up the saved list.

Parameters
itemnameString,
loadOKboolean: false, don't actually load this file, just put it in list (e.g., for name of script when saved).
