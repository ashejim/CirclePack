Detailed Description

This is a linked list for 'Complex' numbers.

This is a linked list for 'Complex' numbers. Such lists are used, e.g., to collect location information from canvas mouse clicks, also from things like circle centers or dual centers, or complex values run through the current 'function' or 'parametric path' in the Function Window.

NOTE: z may be point of plane or a point (theta,phi) of the sphere; conversions are not implemented, so a change in geometry can invalidate these complex numbers.

Mainly, however, process strings of pairs x y of doubles.
