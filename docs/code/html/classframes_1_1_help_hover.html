Detailed Description

Constructs a hover version of the Help frame,.

Constructs a hover version of the Help frame, which holds various help info in tabbed panes. In the future, should add search and other capabilities, improve formating, add tabs such as:

  • functional grouping of commands
  • illustrations (jpg's) of display options.
