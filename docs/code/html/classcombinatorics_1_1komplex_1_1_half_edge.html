Detailed Description

DCEL half-edge: the fundamental directed-edge element of the combinatorial structure.

In a doubly-connected edge list (DCEL), every undirected edge is represented by a pair of oppositely oriented half-edges linked via twin. The face to the left of each half-edge is stored in face. Half-edges around a face are linked by next and prev in counter-clockwise order.

Each half-edge also carries per-edge packing data:

  • invDist — inversive distance (default 1.0 = tangency).
  • schwarzian — discrete Schwarzian derivative.
  • color — display color.
Warning
Half-edges are not persistent across combinatorial modifications (flips, refinements, etc.). Code that caches pointers to half-edges must revalidate after any structural change.
