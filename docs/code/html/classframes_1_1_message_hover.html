Detailed Description

Created at startup.

Created at startup. Provides hovering shell/scratch panel. When locked, panes are scrollable and command entry line is added. Hover occurs when mouse is in "Messages" button or 'Hover' buttons in 'MyConsole's.

This class maintains the command history, for usual up/down command recall. The shell shows command lines as executed, also messages, errors, etc.
