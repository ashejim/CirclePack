◆ sphNormalizer()

Mobius geometry.NSpole.sphNormalizer ( Complex[] pts,
int cycles,
boolean sPole,
boolean debug )
static

Given vector of points in the plane, return a Mobius transformation that puts the centroid of their stereo projections to the sphere close to the origin in 3-space. Return null on error. Note that the resulting Mobius is linear (fixes infinity). If 'sPole' is true, then we include a point located at infinity.

Parameters
ptsComplex[], plane points
cyclesint, iterative cycles
sPoleboolean: include point at infinity
debugboolean
Returns
Mobius, null on failure to converge
