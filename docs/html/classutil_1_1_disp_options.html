Detailed Description

DispOptions simply holds the default display options for a given circle.

DispOptions simply holds the default display options for a given circle packing and supplies it to 'disp' commands when specific flags are lacking. ('disp' and 'disp -wr' calls in 'PackControl.nakedDisp'). The boolean data elements here correspond with checkboxes in the DisplayPanel, 'tailored' is the user-specified text string in DisplayPanel (for advanced users).
