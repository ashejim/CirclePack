Detailed Description

A Frame for sliders controlling specified.

A Frame for sliders controlling specified intrinsic schwarzians s for flowers – see "flower" command in 'SchwarzMap.java'. Note: often computations use the u-variable, u=1-s, see "SchFlowerData", but these sliders controls the s's themselves.

Display will also so the "errors" in the effect of the current schwarzians, see 'SchFlowerData. euclFlower'.

CAUTION: the 'errors' depend on layout process: we start layout using the edge schwarzian closest to the uniform for an n-flower. Starting with a different edge could give different errors. Fortunately, error = 0 is independent of layout process.

Author
kstephe2, June 2020
