Detailed Description

Spherical geodesic data for visual rendering on the apparent sphere.

Spherical geodesic for use in plotting. Set up data for the visual portion of the geodesic only. CAUTION: positions are relevant to the 'apparent' sphere (depends on 'spherical view'). Geodesic considered 'visible' iff:

  • at least one end is ON_FRONT
  • it follows horizon counterclockwise
  • its ends are approximately antipodals ON_HORIZON (so 'lineFlag' is set and use geodesic towards the viewer). Note that 'invisible' geodesic can start or end ON_HORIZON
