Detailed Description

Routines for computing and storing data on a packing.

Routines for computing and storing data on a packing that is a 1-torus. We assume that the radii have been computed. On creation of a TorusData object, the torus will be adjusted to standard normalization with two side-pairings which have been updated.

For flat tori, first corner is at origin, next is at 1, and side-pairings are translations z -> z+a, z -> z+b.

For affine tori, common fixed point of the side-pairing maps is at the origin and the first corner is at 1. Thus the side-pairing maps are z -> az and z -> bz.

There are also static routines for working with tori.
