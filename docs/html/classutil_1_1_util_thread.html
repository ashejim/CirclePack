Detailed Description

In applet/application, the following code will run and stop this thread.

Author
jimrolf In applet/application, the following code will run and stop this thread. 
     UtilThread myThread=null;
     myThread=new UtilThread(jLabel1);
     myThread.start();

     myThread.setStop(true);

Note that I passed in the JLabel jLabel1 in order to see the output. You can easily comment out the jLabel stuff in this class.

The run() method is where you should put code you want to execute.
