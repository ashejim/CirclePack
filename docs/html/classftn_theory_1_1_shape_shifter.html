Detailed Description

The idea is to perturb circle packings with bdry centers on.

The idea is to perturb circle packings with bdry centers on a given polygonal curve; so we want to perturb the curve and be able to adjust the packing so the centers lie on the new curve. This was started with Gerald Orick and Elias Wegert, but development stopped, and the calls to C code will need to be moved to newer C libraries.
