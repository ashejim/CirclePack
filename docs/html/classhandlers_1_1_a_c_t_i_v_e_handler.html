Detailed Description

Active canvasses are those which show packings and allow.

Active canvasses are those which show packings and allow user interaction — toolbars, menus, canvas modes, etc. Currently there are three: the main canvas and the two canvasses in 'PairedFrame'. Note that the static variable 'instance' helps distinguish these for hashing 'MyTool's.
