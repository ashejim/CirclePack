Detailed Description

This is an abstract class encapsulating the various mechanisms for.

This is an abstract class encapsulating the various mechanisms for acquisition, execution, and reporting of CirclePack command strings and for external data sockets. Various sources can be accommodated: 'MyConsole's, sockets, tools, 'fexec' files, etc. (It will be introduced slowly, as time permits and features demand.)
