Detailed Description

This is a geometry utility class.

This is a geometry utility class. It uses a simple matix propogation method to generate 2D points forming a circle or arc of circle.

Start or continue circle list in gp (instantiated Path2D). Cent is Point3D center of spherical circle, {A,B,Cent} is right-hand orthonormal system of Point3D vectors, 'sphrad' is spherical radius. end1 and end2 are Point3D endpoints of the desired arc. If end1=null, want full circle, otherwise we assume end1 and end2 are on the horizon and positively oriented (as viewed from Cent direction).
