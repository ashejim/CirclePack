◆ translate()

NodeLink listManip.NodeLink.translate ( NodeLink nlink,
VertexMap vmap )
static

Return fresh NodeLink with entries translated from 'nlink' using 'vmap'.

CAREFUL: convention is that 'vmap' is {new,old} for translation, so old entry v in 'nlink' and entry <v,V> in 'vmap' leads to new entry V.

If there's no translation for a given v, add 'v' itself to output link.

Parameters
nlinkNodeLink
vmapVertexMap (giving pairs <v,V> for translation)
Returns
NodeLink, new, null if nlink is null.
