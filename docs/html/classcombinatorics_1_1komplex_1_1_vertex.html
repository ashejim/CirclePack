Detailed Description

DCEL vertex holding combinatorial, geometric, and packing data.

Each vertex in the DCEL stores a pointer to one outgoing half-edge (halfedge), from which the full star (flower) can be traversed. For boundary vertices, halfedge points to the downstream (counter-clockwise) boundary edge.

Geometric data includes the circle's center, radius, desired angle sum (aim), and current angle sum (curvature).

Note
In hyperbolic geometry, radii are stored in "x-radius" form: x = 1 − e^{−2h} for finite hyperbolic radius h. For horocycles (h = ∞), the negative of the Euclidean radius is stored for plotting convenience.
See also
HalfEdge The directed edges emanating from this vertex.
RedEdge May override center data for multiply-connected surfaces.
Author
Ken Stephenson (starting 2016)

◆ oppSpoke()

HalfEdge combinatorics.komplex.Vertex.oppSpoke ( HalfEdge spoke,
boolean hexflag )
static

If 'spoke.origin' v is interior, has an even number of spokes, this returns the opposite spoke. If v bdry, this returns the "most-opposite" bdry spoke, bias to downstream bdry edge in case of a tie.

If 'hexflag', then interior v must have 6 spokes and if v is bdry, then if it has 3 faces and 'spoke' itself is bdry, return the other bdry spoke.

(see 'axis_proj').
