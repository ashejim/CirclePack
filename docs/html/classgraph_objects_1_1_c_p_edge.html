Detailed Description

CPEdge. x,y,x2,y2 set from outside. 'drawIt()' is main call. Color settings.

CPEdge. x,y,x2,y2 set from outside. 'drawIt()' is main call. Color settings in graphic context handled elsewhere.

Spherical data is adjusted by parent to act as though sphere is in standard orientation (i.e., looking straight down x-axis toward origin, z-axis vertical).
