Detailed Description

MobiusToolEditor is a popup frame to allow the user to create.

MobiusToolEditor is a popup frame to allow the user to create tools to add to Mobius toolbar. Each tool involves a string of commands for CirclePack, an Icon, and a tooltip. These can be saved in XML format to be read in on startup (going into hashedTools), but the Mobius tools are generally created only when packings are read in or modified.
