String input.QueryParser.queryParse ( PackData p,
String query,
Vector< Vector< String > > flagSegs,
boolean forMsg )
static

Return a String in response to a query. This may have three parts: 'words', 'ans', and 'suffix'. 'ans' may represent (in 'String' form) a value, list, etc, and generally can be used as a variable value in commands.

If 'forMsg' is true, query is intended as a message, so 'words' is prepended. Note that 'words' defaults to "query (p*) = "; may want to change this if it doesn't depend on the current packing (e.g., '?Vlist').

Also, if 'forMsg', then some lists may be limited to 12 items, e.g. if vertex list is longer, put in '...' via a 'suffix' string. Long strings are likewise truncated.
