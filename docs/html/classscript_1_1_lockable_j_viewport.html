Public Member Functions

void setLocked (boolean locked)
 Sets whether or not the viewport is currently locked. While the viewport is locked, it will ignore requests to change the view position. The viewport may be locked multiple times, in which case it will not actually unlock until it is called to unlock at least as many times as it has been called to lock.

Detailed Description

LockableJViewport is a JViewport with the added functionality of view locking. When locked, the viewport will ignore requests to change the view position.
