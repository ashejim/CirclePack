Detailed Description

Slider panels occur in 'Pack Info' under the 'Variables' tab.

Slider panels occur in 'Pack Info' under the 'Variables' tab in a region below the variables table. Each displays its variable name, the slider, slider min/max settings, a 'ftn?' checkbox (e.g. with z*pi to multiply values by pi), slider, current value, and optional command execution button and string.

See 'VariableControlTableModel' in 'VarControl.java'
