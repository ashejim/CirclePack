◆ findContig578()

int[] script.CPTreeNode.findContig578 ( )

Find maximal contiguous sequence of nodes containing (and of same type, either 7==COMMAND, or 8=FILE, or 8=MODE, or 5=LINEUP) as refNode which are closed and in DISPLAY mode. (all these are to be children of same parent.) This is used in creating 'LineSBox's to organize members of this sequence.

Returns
int[4]: [0] set to -1 if type is not 5,7, or 8. Otherwise, these are indices within parent node: [0]=first in contig sequence, [1]=index of refNode itself, [2]=last in contig sequence, [3]=childcount.
