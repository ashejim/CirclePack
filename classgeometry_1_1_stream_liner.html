Detailed Description

Streamline generator for curvature-flow lines on circle packings.

A class for creating streamlines, a la curvature flow lines, as Chuck Collins did using matlab in joint paper with me and Toby Driscoll.

Steamlines are associated with a packing and are represented in 'BaryLink's of of 'BaryPacket's, hence in barycentric coordinates.

The 'basePack' is euclidean, contains the combinatorics and vertex locations. The basePack typically provides a way to plot the streamlines, since the packing used to create them may not have a convenient layout.

◆ whichWay()

BaryPoint geometry.StreamLiner.whichWay ( BaryPoint inpt,
boolean uphill )

Given BaryPoint 'inpt' as starting point, return new BaryPoint with 'face' properly set (i.e., we are determining which face the streamline goes into next). Other routines will find where it goes in that face.

If 'inpt' is interior, return this face or possibly not move at all. If on edge interior, might stay in this face (generally implying a move along the edge to a corner), or go into neighboring face if there is one.

Parameters
inptBaryPoint
Returns
BaryPoint, null on error or no movement (ie. stationary)
