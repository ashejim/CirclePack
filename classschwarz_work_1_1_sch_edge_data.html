Detailed Description

Data for 'HalfEdge's subject to adjustment.

Data for 'HalfEdge's subject to adjustment when circle packing via intrinsic schwarzians. Computing a new schwarzian involves the N-3 previous edge schwarzians, so we want that to be easy to compute.

These objects are instantiate when 'SchwarzPack' is initiated and put in an array, indexed starting at 1. See 'theEdges' for array index.
