Detailed Description

This frame is holds side-by-side canvasses to display 'map'.

This frame is holds side-by-side canvasses to display 'map' type behavior. The user/script can shift between 'PairedFrame' and 'MainFrame' display, not both.

◆ layMeOut() [1/2]

void frames.PairedFrame.layMeOut ( )

This is where the layout is created or recreated (as after a size change).

Overall, its a horizontal Box layout: domBox, midBox, ranBox.

domBox is a vertical Box layout, with domTopPanel, domCanvas, and 'pairConsole'.

ranBox is a vertical Box layout, with ranTopPanel, ranCanvas, and 'movableToolHandler' (which is shared with MainFrame).
