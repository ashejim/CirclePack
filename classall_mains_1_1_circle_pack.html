Detailed Description

Note: all 'CirclePack' actually does is start 'PackControl'.

Note: all 'CirclePack' actually does is start 'PackControl' or 'ShellControl', which extend 'CPBase' and add additional variables.

'PackControl' in turn kicks off various other objects. This is necessary, e.g., so the script frame is ready, and only then is 'startCirclePack' called to load the initial script.
