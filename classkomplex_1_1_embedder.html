Detailed Description

This class contains static methods to embed — more accurately, to.

This class contains static methods to embed — more accurately, to intersect — to the extent possible, one circle packing complex p1 with another p2, starting with designated contiguous vertex pairs in each. The principal routine returns a VertexMap giving <v,w> pairs, where v is a vertex in p1, w is its corresponding vertex in p2 under the embedding.
The result is not necessarily unique — it depends on how the search progresses — and the embedded result may not be a complex, e.g., it may have dangling edges, incomplete flowers, etc. NOTE: This is NOT a sophisticated routine!
